# 新枫之谷（台服冒险岛）的游戏启动方法

由于新枫之谷使用了繁体编码，如果您在简体操作系统直接运行游戏，就会提示：

This client can be executed only in Traditonal Chinese code-page

不用担心，我们只需要知晓启动游戏的方法，就能体验游戏了。

# 没看这一步的人都哭晕了

第一次提示的密码为二级密码，非常重要！

创建角色、进入仓库、现金道具交易等都需要它，尽量简单易记就行，别太复杂！

如果忘记二级密码，您只能邮件联系客户取得帮助，非常麻烦！

游戏角色名一定要包含中文！

创建游戏角色名请一定要包含中文！不然容易锁号！

别起个全英文，或者英文加数字的角色名，一天一锁别来哭诉！

每周三维护

每周三会进入维护，一般是早 8点 到中午 14点，大维护时期具体时间请看官网公告。维护期间是无法进入游戏的！

刚刚注册的账号别急着进入游戏

如果账号是刚刚注册，别急着进游戏，先放松几分钟，不然，会提示下面错误：

等几分钟后（这个过程可能是需要同步港号的相关数据，有时需要隔一两天），就可以进游戏啦！

如果你等一段时间还是提示这个，并且确定进行了两次进阶认证！

那么，您应该重启一下电脑。

另外，好像从 2020年 3月份 开始，就提示为 “请确认您的 Gash 账号与游戏账号是否为正常状态”

# 几种启动方法之间的比较

目前市面大家都使用以下几种方法进行游戏，详细可以通过下面表格对比得出：

对比项原生态网页启动使用 LR 启动第三方缤放登录器启动秋水登陆器TMS小帮手
不用修改系统语言环境✘✔✔✔✔

由于我们大多注册的都是 港号，港号旧的登录接口使用了较旧的技术，仅 IE 浏览器 支持，新的登录接口就支持 谷歌浏览器、火狐浏览器 等内核的浏览器。

  • 原生太网页启动：

    适合单个账号，并不需要记住密码的用户使用，仅需要修改一下系统的语言环境（非常容易修改）就可以做到直接启动游戏并正常畅玩。

  • 使用 LR 启动：

    LE 是 Locale Emulator 的缩写，LR 是 Locale Remulator 的缩写，它能让您的电脑不管处于什么语言环境，都能运行指定语言环境的程序，神器！但是它本身不进行相关账号密码管理，因此，很多单机游戏会用到它。此方法仅仅只是支持游戏程序能正常启动，由于本身不支持账号密码，导致无法登录游戏，眼看着登录界面，却什么也做不了。此方法多用来备用。
    请注意：Locale Emulator 不支持 64 位应用程序，如果您想玩游戏新之谷，请选择 Locale Remulator

  • 第三方缤放登录器启动：

    第三方登录器本身是采用了 LE 来启动游戏（从 4.2 版本开始采用了 LR 来启动游戏），让您可以在简体操作系统里直接启动游戏。并且，由于本身集成了账号管理，可以非常方便进行多账号登录操作。唯一可惜的是：它使用 .Net 开发，配置低的电脑再运行 .Net 开发的程序会占用掉大量的系统资源，并且，如果操作系统不是 Win 10 以上的话，会容易出现一些特别情况。

  • 秋水登陆器：

    全名是秋水橘子香港登陆器。可以很好解决第三方缤放登录器对港号 IP 异常导致的相关出错，由于本身使用 Java 开发，电脑系统需要安装 JRE 运行库，并且本身带有虚拟按键功能，存在更大的封号风险。

  • TMS小帮手:

    小T同学自己编写的小软件，目前还处于开发阶段，等待首个版本发布中。

现在大家会对如何选择适合自己的启动游戏方法有一定的了解，接下来，分析一下他们之间的实现方法，请往下看。

# 方法一：原生态网页启动

要想使用网页启动游戏，你需要把电脑的区域设置为繁体。

打开：控制面板，在控制面板里找到区域。如果你的是 “类别” 显示，你应该先找：时钟和区域，再在里面找到 “区域”

在区域设置的 “管理” 非 unicode 程序的语言里，点击 “更改系统区域设置”，选择设置如下图：

设置繁体区域之后，建议不要使用老版本的第三方缤放登录器来启动游戏啦，不然容易报：C00700C1 错误。

如果您是 Win 7 系统，可能需要重新启动之后才能生效。

TIP

使用谷歌等新一代浏览器访问港号新登录入口页面

登录后启动游戏会提示相关插件安装，已经很友好了，直接按提示安装就可以啦！

如果你无法安装谷歌浏览器插件，你可以使用 EDGE 浏览器 或 火狐浏览器，就可以直接安装插件啦！

Windows 11 打开 IE 浏览器的方法

使用 IE 浏览器访问：港号 beanfun 官网

如果您的操作系统是 Win 11，您可能找不到 IE 浏览器，不用担心，您只需要按下面步骤操作，就可以打开 IE 浏览器。

第一步：打开 控制面板

  • 您可以按 Win + R，在运行界面里输入：control
  • 您可以按 Win + E，在资源管理器地址栏里输入：control、或者 控制面板

第二步：打开 Internet 选项

如果您当前的查看方式是类别，您可以点击：网络和 Internet - Internet 选项

如果您当前的查看方式是大图标或者小图标，请直接找到 Internet 选项

第二步：打开 IE 浏览器

在 Internet 选项 界面，直接点击右上角的 ?，就可以打开 IE 浏览器了。

友情提示

可以使用 LR 转区启动浏览器，这样就可以在简单操作系统里，不用修改系统区域语言，也能使用网页启动游戏啦！

# 方法二：使用 LE/LR 启动

由于 LE 不支持 64 位应用，现在都是使用 LR 来启动游戏。

如果仅仅只使用 LR 来启动游戏，是无法进入游戏游玩的，多数高级玩家是使用 LR 转区启动浏览器，然后配合网页启动游戏。

TIP

MapleStory 全面启用 64 位应用程序，LE 已经无法支持，如果需要，请使用 LR 来启动游戏。

如果使用 Locale Remulator 启动游戏时出现一些问题，您可以安装最新版的微软 VC 运行库以解决相关问题：

下载 VC x86 运行库 ｜ 下载 VC x64 运行库 两个都安装之后就可以正常使用了。

# 方法三：第三方缤放登录器启动

WARNING

由于 MapleStory 全面启用 64 位应用程序，您需要使用以下版本或更新版本的第三方缤放登录器：

  • 4.4+

如果想使用第三方缤放登录器来启动游戏，请更新到最新版本！

旧缤放登录器依旧使用 LE 模块来转区启动，暂时不支持 64 位应用程序。使用旧版本启动游戏将会提示：以非繁体语系系统启动游戏的方式不支援64-Bit游戏。

如果出现无法加载 DLL 此问题，您可以安装最新版本的微软 VC++ 运行库来解决此问题：

  • 1、下载 VC x86 运行库 ｜ 下载 VC x64 运行库 两个都安装之后就可以正常使用了。
  • 2、前往 Github 下载所有的微软运行库，这个是包含了全部的 VC 运行库，可以同时解决依赖其他版本的软件正常使用。

更多问题解决请查阅：缤放登录器-乐豆第三方客户端：使用教程

DANGER

请不要选中 自动登录，不然，很容易出现卡死、启动无界面、闪退、提示模块等等错误！

使用 “第三方缤放登录器” 可以很好解决 “This client can be executed only in Traditonal Chinese code-page” 这类问题，因为 “缤放” 已经引用了 Locale Emulator 模块，简体系统，一样玩新枫之谷！（从 4.2 版本开始使用 Locale Remulator 模块来启动 64 位应用程序）

前往：第三方缤放 Github 下载页面

游戏账号需要在 第三方缤放登录器 软件界面里双击自己的账号名称，就会复制一段类似乱码一样的字符串，这就是游戏的账号；密码需要点 获取密码 才会出现。

如果你的账号是刚刚注册的，登录缤放之后，是无法获取密码的，你还需要进行下列操作：

1.在 第三方缤放登录器 主界面右下角，点击 “新增账号”

2.在 “使用者名称” 里输入注册账号时填写的昵称（其实可以随便填的，但是发生过 BUG，这个 BUG 应该不会再出现了。）

如果您无法新增账号，您需要使用 IE 浏览器，在浏览器里网页在线增加！

3.如果你的 第三方缤放登录器 底下没有获取密码按钮，那么你需要在 “设定” 里选中 “传统登入模式”

4.新版本 第三方缤放登录器 默认是隐藏了账号，你可以在账户名称里双击复制你的账号。（如果选中自动输入，获取密码后，第三方缤放登录器会自动帮你输入账号和密码）

# 进入游戏后闪退

TIP

很多加速器加速效果并不理想，也会导致进入游戏界面时闪退！

如果是此情况，建议选择白嫖的免费加速器：2个n点com（由于没有收到此加速器的推广费用，不知道会不会给我打钱）

如果您启动游戏之后，默认首次是全屏显示游戏，并且是刚进入全屏时就闪退，这时，您应该考虑是显卡驱动问题。

您可以尝试下面方法来临时解决此问题：

  • 刚进入全屏是，快速按一次组合键：Alt + Enter(回车键)，切换到窗口模式进行游戏。

如果您想解决全屏闪退的问题，您需要不停尝试不同版本的显卡驱动，直到一个稳定支持全屏游戏的驱动版本。
