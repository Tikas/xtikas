# 新枫之谷游戏的基本操作【新手入门篇】

对于一个刚接触 MapleStory 这款游戏的人来说，有一个入门教程是非常友好的，对此 小T同学 决定撰写一些入门上手文章，以方便新人们能快速从零开始上手这款游戏。

# 写在前言

MapleStory 这款游戏是有着非常多的职业，而很多职业一开始都是从职业群共有的新手村开始，随着等级的提高，可以选择不同的职业进行转职，其中 冒险家 这个职业群最为古老，从游戏之初就活跃在 MapleStory 各个角落，他们才是支撑起这游戏的一群职业。

虽然 MapleStory 经历过大巨变等等大改动，然而，枫之岛（彩虹村）依旧是许多人心中梦的起点，让我们一起追随前人们的脚步，从 冒险家 开始体验游戏的基本操作。

TIP

为了您能有一个良好的游戏体验，请务必完成冒险家的基本操作之后，停止冒险家的旅程！

冒险家对新手非常不友好，我们只需要体验一下游戏的基本操作，之后就停下冒险家的冒险旅程，更换其他对新手更加友好的职业进一步游戏体验。

# 一、创建冒险家角色

进入游戏之后，您或许可能需要等一会，创建角色 按钮才会激活，如果您当前的版本有 斗燃 活动，会在 创建角色 按钮上面显示 “可创建燃烧角色 1个”，刚开始玩，先以了解为主，请不要创建斗燃角色。

如果您是第一次创建角色，会提示您设置您的二级密码，这个二级密码非常重要，建议简单易记就可以。

验证码无误，不出意外，您会进来 选择职业 界面，如下图所示（不同版本其显示的职业排序不同，但界面操作相同）

中间最下面一栏所显示的职业，并没有显示出全部职业（或职业群体），您需要点击 〈 、〉 来展示其余职业（或职业群体），尝试找到 冒险家 并点击 新创角色 来创建一个初心者吧（冒险家职业群最开始时职业名称），角色名要带有中文哦！出现 设定按键模式 时，先别选择，可以看接下来的第二部分。

TIP

角色名一定要带有中文哦！

小节总结：
A：学会进入创建角色的界面
B：学会选择指定的职业或职业群进行创建角色

# 二、按键模式选择

当您输入的 角色名 合法并且没有被占用时，就会提示 设定按键模式：

其中两种模式大概介绍如下：

  • 基本模式设定：适合老玩家使用。老玩家从最初的游戏版本开始冒险，那时候默认统一使用的是这个按键模式，如果您是老玩家，并且已经习惯了旧的按键方式，可以选择此模式。（注：此模式布局非常不合理，如果您并非老玩家，请不要选择此模式）
  • 进阶模式设定：此按键模式，非常适合新玩家使用，相信看此教程的您，应该是新玩家，不用犹豫，直接选择此模式吧！

如果您当前的游戏版本有斗燃活动，在您选择按键模式之后，您还能再看到一个提示是否设定此角色为燃烧角色，请点 否

TIP

如果不小心创建了燃烧角色，您可以删除此角色，等 24小时 后再次确认删除，就可以重新创建新的极限燃烧角色了。

游戏内可以随时更换按键模式

您不用担心选错按键模式这个问题，因为您可以在游戏内随时更换按键模式（说是随时，其实还是有要求，您在一些特殊场景时，是无法进行按键更改的）。

小节总结：
A：了解两种按键模式的区别，并选择适合自己的默认按键模式

# 三、学会角色移动操作

进入游戏，您会出现在一棵大枫树旁边，并且前面还有一位小姐姐向着右边远去。

这时候，您已经能看到 MapleStory 的整体游戏风格，您的角色应该是在游戏画面的左边，而之前出现的小姐姐也是指引您向右边行走，这也是一个贯穿了绝大部分情景的一个要点：

左走回去，右走前进。

这个要点在您做一些任务、剧情、BOSS挑战等等时候，几乎适用。

而您，可以尝试使用键盘的 ↑上、↓下、←左、→右 四个按键对角色进行操作，大概尝试结果如下：

  • ↑ 向上键没有任何反应
  • ↓ 向下键角色趴在地上，一直按住，就一直趴着
  • ← 向左键角色会向左移动一下，一直按住，就一直向左移动
  • → 向右键角色会向右移动一下，一直按住，就一直向右移动

如果您尝试结果如上所说，恭喜，您已经学会移动角色了。

您可以一直向右边走，能看到一个散发光芒的 过传送点，在这里按 ↑ 向上键，就可以进入下一个地方。

小节总结：
A：记住游戏的一个要点：左走回去，右走前进。
B：学会使用 ↑上、↓下、←左、→右 四个按键对角色进行操作。

# 四、学会任务交接、技能加点

从枫叶山丘（大枫树）走到前面一个地方，您能看到 麦加 ，和她对话，在任务的多选择里，选择默认的第一个不跳过，之后您的角色会升一等级，然后进行学习一个重要的技能加点，尝试如下图所示例，把您学习的技能放到 键盘快捷键 里，如果您不知道怎么打开 键盘快捷键，请接着往下看：

当然，刚玩的您，想来还不知道如何打开 键盘快捷键 ，请放心，作为入门教程，这就为您讲解如何打开 键盘快捷键：

  • 1、默认情况下，您可以尝试按 \ 键打开（再按一次关闭） 键盘快捷键 设置窗口（注：\ 并不是 / 这个键，/ 这个键上面一般会有一个 ？ 号）
  • 2、您可以多次按 Esc 键，直到出现 SETTING 菜单，一般当前默认是选中 频道变更，您可以尝试向下键选中 安装快捷键 之后，使用 回车键 来确认打开 键盘快捷键 设置窗口

认真的您，一定会发现 K 键对应之前出现的技能面板，尝试一下学习一个技能，并把它放到 键盘快捷键 里，确认 保存按键设置之后，再尝试按设置好的键使用一下吧！

TIP

多数情况下，键盘快捷键里的相关功能面板，都可以通过指定的按键进行打开面板或关闭面板。

少数如 灵魂武器、指导 面板，需要手动关闭。（指导面板使用按键默认是展开/收起，但是，可以使用 Esc 键来关闭）

接下来再同 麦加 对话，正式接取您的第一个任务：任务开始 。

别急着一直按回车键或空格键，您可以好好看看对话内容，这些都是新人需要掌握的知识。

对话完，您就可以完成这个任务，快同 麦加 同学领取 轻松椅子 吧！

恭喜，您已经初步学会领取任务和完成任务，接下来向右边前往下一个地方吧

小节总结：
A：学会使用按键打开相关功能面板（如 键盘快捷键、技能 等面板）
B：学会加点技能并设置按键后使用技能
C：学会任务的交接

# 五、学会跳跃障碍、狩猎怪物、拾取物品、使用消耗品

现在您已经来到了枫之岛的小树林，好吧，虽然看不到几棵小树，但是这个图名字就叫小树林，不用在意细节。

麦加 同学这次会教您狩猎，让您增长经验，成为强者。

而 麦加 同学只告诉您使用 Ctrl 来攻击，并没有告诉您如果跳跃障碍，您可以打开 键盘快捷键 查看那个按键是 跳 功能，不出意外，您会发现 Alt 是弹跳键。

尝试单独使用 Alt ，再尝试按住 → 向右键不放，再按一次 Alt

尝试其他不同的按键，试试看各个功能的面板，比如后面升级比较常用的 能力值 面板，每个按键都尝试去了解一下吧！

如果您能跳到前面的小高地，恭喜，您可以使用 Ctrl 来攻击小石球，不出意外，地面上会出现物品，此物品上面会有按键提示，尝试角色走到物品里，使用此按键拾取物品吧！

TIP

后期有威力更强大的技能时，请使用技能来顶替 Ctrl。

完成 麦加 同学的狩猎任务，接着接受新的任务：吃东西的话就会恢复

请您找到 道具 功能的按键，并在 道具 面板里找到 消耗 里的 苹果，尝试双击使用吧！

道具面板介绍

在 道具 面板里包含有装备、消耗、其他、装饰、特殊、时装，这是一个使用频率非常高的按键，每个道具栏分类的具体物品如下：

  • 装备：这里是存放装备的地方，像图腾那些道具也是存放在装备栏里（如：轮回碑石等）。
  • 消耗：这里存放了能消耗使用的物品（如：药水、经验加倍券、怪怪卡（萌兽）、轮回星火等）。
  • 其他：这里存放了一些怪物掉落物品、任务物品、相关硬币、相关材料（如：矿石、草药等）等等，他们是不能直接使用，但是可以通过一些 NPC 或相关功能面板进行使用，相关物品或材料还能放拍卖（或 NPC）卖个好价钱。（注：NPC 指游戏里的人物，非玩家的角色）
  • 装饰：好吧，这里其实更多是存放椅子的。
  • 特殊：这里多放宠物相关。
  • 时装：好看的时装道具就在这栏啦，燃烧之戒也是在这里存放的哦。

如果您后续玩的角色是大号，不妨从拍卖行里购买下面扩展道具，非常实用哦：

非常实用的扩展道具是：製作法20格背包、卷軸20格背包、製作物品 20格欄位背包、粉紅色硬幣錢包（名称使用繁体，可以直接复制去拍卖购买） -- 先记下来。

当然，背包并不止上面所列出的四样，还有更多，您可以尝试搜索发现吧！

接下来 麦加 同学会告诉您，消耗栏里的恢复药水也是可以同技能栏里的技能一样放置到 键盘快捷键 里，以方便快速使用。

小节总结：
A：学会跳跃，并且灵活搭配左右方向键来跳跃
B：学会狩猎怪物并拾取掉落的物品
C：学会消耗品使用方法，后续有机会尝试如同技能那样放置到键盘快捷键里，以方便使用。

# 六、学会爬绳、跨过怪物、更换装备

再次向右前进，您会来到枫之岛的大菇菇这个地方，这里有着非常多的小蜗牛，是一个很适合练习跨跃怪物的地方，这个技巧在您完成一些特殊地图的任务时，非常有用。

同样，这里也有非常多的绳子可以尝试使用：跳 + ↑ 向上键 来爬绳子（或者其他地方的一些梯子）

TIP

尝试接近绳子时，使用向前跳跃，并在空中按向上键，来爬绳子。

尝试不对怪物进行攻击，使用弹跳键进行跨跃怪物练习

完成 麦加 同学的跳跳跳任务，您能得到新手礼物：两件装备及一些药水。您可以尝试打开装备面板（和麦加同学的对话，也会有提示），然后在背包的装备栏里双击新装备试试看。如下图所示，那么，恭喜您，您已经学会更换装备了：

之后就可以前往枫叶村会见村长啦，冒险家的旅程就可以先告一段落了，因为接下来就是离开新手村啦，由于冒险家转职是有五个方向，这个角色就可以放一边，等后续确认自己以那个职业为主号（又名：本尊、大号）时，再选择一些强而有力的联盟属性或者 Link 传授加成的职业。

您可以按 Esc 键，在出现的 SETTING 菜单里选择 结束游戏 来返回角色选择界面，当前，在您返回之前，您可以先尝试本文后部分的键盘设置教程。

小节总结：
A：学会跳跃跨过怪物
B：学会爬绳
C：学会更换装备

# 七、设置符合自己的按键

由于每个人的按键习惯及手掌大小不同，加上每个职业的技能形态也不相同，在按键布局上同样也会不相同，在此，我就不放出自己的按键设置，只从科学合理的角度加上一些人个心得，来指导您如何设置符合自己的按键。

而 MapleStory 这款游戏多数主要是打怪升级，一个不符合自己的按键会很容易让自己的手累坏。

# 7.1 先学会调整按键

您可以在 键盘快捷键 里，鼠标左键单击以选中相关按键设置，然后移动到别处再鼠标左键单击，放置按键设置到新的位置：

您也可以鼠标左键双击以取消此按键设置：

当您按键改动的很乱时，您可以通过 初始化 来恢复到游戏默认的按键布局。

接下来好好设置自己的按键吧！

# 7.2 以舒适为主

首先，您可以以平时打字之前，左手放置在键盘上，然后右手放置在上下左右四个方向键。

不出意外，您的右手食指、中指、无名指分别放在向左键、向下键、向右键，您的左手食指会放在 F 键上，而大姆指很自然落在空格键上。

现在由于左手比较向前伸，相对右手来说，她们位置处于不平行状态，先长时间感受一下这时的坐姿手势，是否舒服。

如果不舒服，请调整一下，直到找到自己认为能长时间保持舒适的手势。

# 7.3 模拟常用的游戏按键

您可能还没有深入了解游戏的每个职业特色，但是前面已经学习了游戏的基本操作，算是有个基本了解，为此，小T同学 在这总结一下游戏的常用一些操作按键：

  • 不可调整却必需用到：
    • ↑上、↓下、←左、→右
    • 回车键
  • 可调整，为保持操作习惯尽量不更改：
    • Esc键
    • Tab键
  • 使用频繁的功能/技能
    • 弹跳（多数法师还要增加位移技能）
    • 主输出技能（无CD）
  • 次级使用频繁的功能/技能
    • CD 时间十来秒的技能
    • 带来相关可观收益的技能（如五转的通用位移技能或其他爆发类）
    • 採集（多用在任务交互、解地图轮为主，如果您习惯使用鼠标，它的使用频率并不高。但是，当您经常副业挖矿采药时，使用频率直接上升）
  • 一些常用 BUFF 加持技能
    • 常驻的放宠物里自动释放
    • 次要的可以设置宏按键节省位置来释放（或者使用内面耀光道具自动释放）
    • 只能手动释放又不能不放的、或者加成可观的（如一些开关型 BUFF 技能、五转的经验加持 BUFF 技能等）
  • 不常用却需要的功能/技能/物品
    • 捡取（平时是宠物自动拾取，但是宠物不会拾取玩家丢出的道具）
    • 道具（背包面板，强化分解等情况也需要）
    • 能力值（属性加点）
    • 技能（有时切换技能核心或者相关道具时，需要去 0转 或 5转 里拉取出来）
    • 指导（有它可以快速去一些完成印章的地方）
    • 装备窗（这个可以查看装备、设置宠物是否拾取，以及时装相关等等）
    • 攻击（最开始 Ctrl 键的普通攻击，这个键在一些特殊场合需要）
    • 等等（未列举就需要您在游戏内探索发现）

从上面分类得出，游戏按键分为：位移、技能、BUFF、功能按键、物品这几大类，而游戏多是处于打怪升级或者挑战 BOSS，所以位移和技能及一些加成 BUFF 就显得很重要了，BUFF 由于是持续性，并不是一直需要使用，我们暂时先不考虑，故，最为常用的是位移和技能。

所以，您可以模拟一下使用位移按键、主要输出技能（无 CD）、次要输出技能（CD 快），以确定自己最常用的按键位置。

如：

  • 空格键为跳
  • C 键为跳
  • 保持 Alt 键为跳
  • 等等等...

TIP

波浪键一般默认是 购物商场，这个功能完全没必要占用一个按键设置，真需要时，可以通过点击商城图标来进入。

小T同学 一般是把波浪键设置为 捡起 键，或者是宏键以释放 BUFF 用。您可以参考参考。

小节总结：
A：学会按键更改设置
B：尝试设置符合自己的按键习惯
