# 新枫之谷（台服冒险岛）锁号的解锁教程

当您看到 “账号是否为正常状态” 时，请不要紧张，这多是您的号触发了反外挂机制。

我们只需要和客服联系一下，发个电邮联络一下（台号可以打个电话），就可以等待客服跟进处理完，就可以解除 “确认你的 Gash 账号与游戏账号是否为正常状态” 了。

DANGER

由于港号太多人利用社工库获取的个人信息，联系客服进行账号修改，导致非常多的盗号事件，影响之深远。

现在港号使用了全新的客服处理系统，解锁等相关账号的操作，需要提供以下内容：

  • 1.橘子的平台账号
  • 2.新枫之谷的游戏账号（就是在个人中心里的 BFM 那个账号）
  • 3.进阶认证的手机号近一两个月的账单信息，建议去营业厅打印出来，并盖上营业厅的店章或相关章
  • 4.最好进手机的营业厅 APP，开对应月份的电子发票，并提供此电子发票给橘子。如果可以，也提供一下 APP 里话费账单截图或开票的截图
  • 5.个人证件的正反面

现在解锁需要的资料过于繁多，请大家注册账号时信息一定要真实。

以下为旧的文章内容，可能不适用于现在。由于本人的忙碌，非常抱歉没能及时更新新的内容，给您带来不便，非常抱歉！

# 一、为什么会被锁号？

新枫之谷的账号锁号了，账号会显示 “灰色” 的状态，尝试进入游戏，会提示 “确认你的 Gash 账号与游戏账号是否为正常状态”。

多为下面几个原因导致：

  • 1.通常账号被锁，是保护账号安全的一种措施，这多是切换 IP 频繁，异常操作过多，系统会自动帮您的账号锁定，以保护您的账号安全。 当然，这情况多是在刚刚注册新账号不久的用户（注册不到半年的账号）。
  • 2.账号没有进行进阶认证也会提示这个，请进阶认证之后尝试，就可以正常进入游戏。
  • 3.触发反外挂机制，系统需要核实您的账号信息，这情况下，也会锁定。只需要提交解锁申请就可以得到解锁了。
  • 4.游戏内角色名采用全英文名，或者英文加数字组合，没有任何一个中文文字，这情况也会频繁被锁！ 因为许多工作室都采用这种命名方式，反外挂系统很容易误判您的账号为工作室账号。
  • 5.购买的道具异常（黑货），当号主找回时，你的账号会被锁定，直到调查完异常道具前后来源之后，才能得以解锁。 如果您在这期间，不能提交相关证明，您购买的道具将会清除！
  • 6.使用黑卡储值卡进行充值时，也会被锁定账号，直到补全黑卡的点数，才能解锁。
  • 7.待续......可能还有其他没补全的，暂时就上面这些吧。

什么时候会触发反外挂机制呢？

以下情况，会触发反外挂机制：

  • 使用非法外挂程序
  • 使用按键程序，如按键精灵
  • 使用鼠标连点
  • 长时间一直按一个键，卡键等
  • 和使用外挂的人组队获利也容易触发反外挂机制
  • 加速器同一节点有人触发反外挂机制时，可能会连带检测

WARNING

如果您的账号是刚刚注册，您应该是还没有进行进阶认证导致的。

# 二、查询锁号的原因

查看被锁原因，可以点 “会员中心” - “查询记录” - “大事件” 里查看。

选择当前月份，再选择 “枫之谷” 游戏，点击 “查询” 就可以显示出锁号的原因。

之前的大事件是不能查询当天的，最早是昨天的，现在不知道能不能查询当天的信息了。

# 三、台号解锁方法

台号只需要拨打 +886 2-21926100

（如移动可以使用 12593 ，在 886 前面加多两个 0，如：12593 00886 221926100，不用空格，这里的空格只是为了方便阅读。）

实际披打号码示例：1259300886221926100

不是账号进阶的电话号码也能解。

打通电话之后说明自己的被锁账号，客服会验证您绑定的电话号码，以及绑定的邮箱。

信息核对无误就可以解锁啦。

当然，如果道具异常之类的，就需要您提供道具来源信息了。

# 四、港号解锁方法流程

从效率上来说，并没有台号那么快。一般是半小时到两天左右。

如果您的账号是被锁状态超时很久，账号内会被清空（官方回复说不会清空，但是事实就是里面角色都没了）

如果是由于使用黑卡导致，请补全负数点数之后，填写已补全就可以了。😦

# 五、港号解锁步骤同样适用于其他需要联系客服的情况

港号的解锁方法，这个步骤还适用其他需要客服解决的情况，如：

如果您需要修改二级密码，可以发此申请表给客服，填写内容为：需求更改游戏二级密码，您不在游戏的时间段，以及您在游戏的时间段。

比如：客服大大，我需要修改二级密码，早上 10点到 12点不在线，中午 12点后就会在线，有劳了。

当然别忘记说要改成的二级密码是什么，不然，客服都不知道：到底是要那个密码做为二级密码。XD

TIP

如果您忘记了自己的密码，连保安问题的答案也忘记了，您就不能使用这个方法来修改账号的密码了。

这时，您可以给香港 beanfun 客服致电寻求解决办法，电话：(852) 37115511

例如移动手机可以披打：125930085237115511

一般是需要您下载 “香港游戏橘子客服传真表” 填写好并传真过去，当然您可以同客服沟通好以扫描图片方式发邮件过去。

香港橘子客服传真热线：(852)2169-8799

当然，也可以另外申请一个新的账号，然后用新的账号发送邮件进行取得帮助。
