# 新枫之谷（台服冒险岛）账号注册教程

DANGER

最近港号发生大量账号以 QQ 号邮箱的盗号事件，其初步被盗价值已超 1000万元，事件影响之大，请大家一定以本文步骤去注册账号，以保护自己的账号安全！

盗号者通过人们惯用的懒记忆，以 QQ 邮箱为突破口，收集各个交易群、交流群、等等群，以收集到目标 QQ 号，然后进行反向查询绑定电话查询，之后进行电话实名查询，得到目标的游戏账号邮箱、电话号码、身份证信息，再通过相关库和黑链可以进一步得到更多个人信息。

盗号者们收集齐信息之后，就开始作案，准备好仿冒的个人信息，先申请更改账号邮箱，修改二级密码，在原主不知情的情况下，直接整个账号所有权转移，之后就是清号。

保护自己的账号安全，从注册开始，请跟随 小T同学 的注册教程，留意 重要安全防护要点，小T同学 将全程安全守护您的账号！

# 认识账号类型

新枫之谷的注册账号可以分为两部分：

  • 1.注册 beanfun 平台账号
  • 2.进阶认证 beanfun 平台账号

本文以注册港号为实例

TIP

beanfun 平台账号 并不是游戏账号！

新枫之谷（台服冒险岛）是采用先登录 beanfun 平台，再启动游戏。

游戏账号需要在 beanfun 里新增时生成，是一串类似 MD5 的值。

游戏的密码采用动态密码，每次从 beanfun 里启动游戏，密码都不一样。

# 1️⃣ 正确认知港号和台号

由于港服和台服已经合并。现在的新枫之谷，是同时支持港号和台号在线游戏的。

港号和台号，可以通过下面表格进行对比

对比项港号台号
对 Email 泛支持✔✔
+86 手机号码✔✘
内地身份✔✘
邮件解锁✔✔
电话解锁✘✔
双游戏账号支持✘✔
QC扫码登录✘✔

由上面表格分析可以得出，注册账号是以 港号 为准，实际玩游戏时，是去 台号 所在的服务器。

而本文也是为 港号 注册而撰写。

# 2️⃣ 以下情况请不要进行账号注册

很多新人刚来台服，对台服环境并不了解，作为指导的我，在这里建议以下情况请不要进行账号注册：

  • A）如果您是在深夜注册的，可能会出现账号一切正常，获取密码的时候，一直提示账号服务不存在！请别在深夜注册账号！深夜时段：00:00 - 08:00

  • B）如果您是在维护期间注册账号，也可能会出现提示账号服务不存在！请别在星期三的 00:00 - 15:00 时段进行账号注册！

  • C）无法避免的特殊情况，如：

    • 每周三例行维护时导致港号数据库不能及时同步到游戏服务器（建议等一两天）
    • 大量工作室涌入，导致新注册的账号暂时无法取得密码（一般是短期性几天，很少见）

如果您真的提示了：“服务账号不存在”，可以暂时等待第二天尝试（一般是 00:00 后会进行服务器数据库更新，2022年3月有小伙伴反馈同步数据库需要两到三天，账号才能正常使用），如果还没恢复正常，您需要查阅 【解锁教程】联系客服 进行账号异常处理。

# 3️⃣ 注册账号（港号）

请确保您访问的注册页面是 hk.beanfun.com 域名，以防止被镜像网站知晓您注册的账号信息！

前往>>：港号新注册页面（建议使用 Chrome 浏览器打开，需要验证 Google reCAPTCHA，如果没有相关，可以使用 Watt Toolkit）

在注册时，填写的一些内容会属于超级重要！

请大家牢记这些重点：

  • 電郵：这是您使用的电子邮箱，也是您的 beanfun 账号，同时，也是需要您接受验证或相关重要事项时，会发邮件到您注册时填写的这个邮箱。

    注意：建议您不要使用常用的 QQ 邮箱进行注册，以防止被他人社工盗号。

  • 設定暱稱：这个简单一些，没实际意义，此昵称并非账号！建议使用中文，请不要设置为账号相关的信息！
  • 設定密码：由于游戏内道具都是非常值钱，建议您使用一个全新的复杂密码，从未在别的地方使用过的那种，还要不会被容易社工猜测出来的那种！

    注意：请牢记您的密码，如果忘记又没有采用第三方登录器之类有记住密码功能的，修改密码将是一件相对困难的事情（前提您记得保安问题的答案）。
    如果您连同保安问题的答案也忘记，此时修改密码将会是一件“非常困难”的事，因为您需要重新注册一个新账号，用新的账号发起邮件联系，还要提供一堆的相关证据。实则如是：难上加难！

  • 出生日期：请和您的要求保持一致，以方便您的账号能顺利进阶认证。
  • 性別：请和您的身份保持一致，以方便您的账号能顺利进阶认证。
  • 保安問題 和 答案：这个非常重要，注册页面也会有重要文字提示，请牢记您选择的问题及答案！

    注意：如果您忘记保安问题和答案，您可以在能正常使用账号时，发起邮件联系客服，提出修改这两个，此时，您需要提交相关对应的表格及身份。

在填写验证码和同意条款提交之后，就算完成注册啦。当然，并没有完全注册完，因为您还需要前往自己的邮箱验证，验证完之后，才算是初步成功！

如果您需要注册台号，您可以从下面链接前往注册：

前往：台号注册页面（建议使用 Chrome 浏览器打开）

重要安全防护要点：一

电邮是 beanfun 的账号，也算是管理游戏的账号，请不要使用 QQ 号码的邮箱，非常关键！！！

很多人的账号被盗就是使用容易被社工的 QQ 号码的邮箱注册 beanfun 账号！

如果您一定使用 QQ 的邮箱，建议您设置一个英文的邮箱账号，并且，不要设置为默认发信账号！！！

您可以在 QQ 邮箱里的 【设置】-【账户】-【英文邮箱账号】。

设置时尽量长一些，以达到从 beanfun 里的会员信息显示更多【 * 】号，如果能英文 + 数字结合更好（请避开容易被猜测的个人信息为英文的邮箱账号）

重要安全防护要点：二

设定昵称尽量是中文，并且和个人信息没有关联！

更不能和账号密码相关联！！！

重要安全防护要点：三

由于密码最大只能设置 20位，字母和数字组合，如果可以，请您使用相关密码管理器，使用随机密码。

如果没有使用密码管理器，可以使用一段中文进行多种编码加密，最后可以只取最大密码长度 20 位。

更详情的安全知识，请查阅此文：多招教您防护自己的账号安全

# 4️⃣ 进阶认证

之前只是注册了 Beanfun 账号，未进行进阶认证的 Beanfun 账号，是无法进入游戏的。

SO，赶紧来认证吧！

# 4.1 安装官方缤放平台服务程序

TIP

前往：Beanfun 外挂程序下载页面（无法访问请打开加速器或者关闭加速器）

无论通过那种途径进行认证操作，都需要使用到官方缤放平台服务程序！

由于官方缤放平台服务程序和玩家常用的 “第三方缤放登录器” 属于同名同图标，不易区分。
为了容易区分，本文为官方起名的 Beanfun 外挂程序 改名为：官方缤放平台服务程序。
而 “第三方缤放登录器” 依旧使用此名，以加强此 “缤放” 是第三方开发，并只作用于登录使用。
注：第三方缤放登录器从 V4.X 版本开始，使用和官方不同的图标以进行区分。

等待 官方缤放平台服务程序 的安装完成，直到出现 “正在更新档案”，并且 更新完成。

如果未进行更新档案，或者更新出错，请手动运行一次桌面的 beanfun！快捷方式，或者重新安装一次官方的 beanfun 外挂程序，进行再一次安装。
一定要在 “正在更新档案” 里出现 更新完成
如果还是无法更新完成，请换别的加速器，或者关掉加速器，直到出现更新完成界面。

# 4.2 两种途径前往进阶认证

这里列出了两种常见的进阶认证方法：

一个是官网操作，安全有保障，相对来说就是麻烦了一些。

另一个是使用 “第三方缤放登录器” 进行操作，方便简单，由于是第三方产品，安全上相对来说有风险性。（开发者已经开源，感兴趣的朋友可以去查阅源代码）

由于港号使用了较旧的技术，只支持 IE 浏览器。

如果您使用网页（通常把不使用第三方登录器称为使用网页）来登录账号进行游戏的话，您还需要额外设置一下 IE 游戏器：

第一步：去掉 IE 浏览器的保护模式

打开 IE 的 Internet 选项，在安全里去掉 “启用保护模式”，保存并关掉 IE ，重新打开 IE 就生效了。

如下图所示：

第二步：登录注册的账号，并进行进阶认证

打开 Beanfun 首页

在首页的右下角，登入刚刚注册的账号。

然后，在同样的右下角里，点击 會員中心 打开个人信息以方便进行进阶认证。

TIP

第三方缤放登录器是一个非常好用的工具。然而，由于并非官方发布的，也有不法人士进行对他修改捆绑了盗号程序，让开发者非常难过。

如果您同我一样，非常喜欢使用登录器玩游戏（体验感非常好），建议您一定要去作者官方的 Github 下载！

前往：第三方缤放 Github 下载页面

第三步：打开 IE 浏览器

# 5️⃣ 使用 BF 生成的 ID 编码作为账号

当您注册好账号，别急着记住密码以方便自己，您可以完全放弃使用邮箱的账号去登录，而使用更安全的 ID 编码作为账号登录。

这样在登录时就不会把自己的邮箱地址暴露出去。

要想拿到这个 ID 账号，您需要申请修改一次密码，可以不改密码。

重要安全防护要点：四

如果您的账号真的需要长久分享给朋友，请您只发送此 ID 编码作为账号，这样对方的第三方缤放登录器选择切换其他账号时只显示此 BF ID 账号，防止截图、录像等泄漏出去。

这样能有效防止别人知道自己的账号是邮箱 ，以防止被别人社工盗号！

注：登录之后可以从联系客服里查看完整的邮箱地址。

每一个账号，都有一个 BF 等同的 ID 作为账号，您可以通过以下步骤，查看到自己的 BF ID：

# 5.1、第一步：为账号变更密码

建议使用第三方缤放登录器操作，更为方便！

以下是以 IE 浏览器操作为演示，如果您使用第三方缤放登录器，可以不需要 IE 浏览器。

如果您是 win 11 系统，您可以参考前面的【Windows 11 打开 IE 浏览器的方法】

使用 IE 浏览器访问 Beanfun 官网 登录一下。

登录账号之后，点击【会员中心】，点击【变更密码】，输入您注册时的邮箱，将会收到修改密码链接的邮件。

# 5.2、第二步：访问重新设定新密码页面

建议使用谷歌浏览器，或者使用火狐浏览器访问邮箱，点击邮件里的链接地址。

别问我为什么是这个颜色，真是服了橘子的审美观！

在重新设定新密码页面，查看修改密码页面的源代码，您可以使用 Ctrl + U 或者右键选择 查看网页源代码，如果这两个方法不能查看到源代码，您可以直接按 F12。

在源代码里，按 Ctrl + F ，搜索这个内容：id="__VIEWSTATE"

复制 value 的值。注意：请不要复制引号，仅需要复制引号里面的内容。

注：我们只需要访问此重新设定新密码页面即可，无需修改新的密码，当然，如果您刚好想修改密码，也是可以的。

# 5.3、第三步：解码 value 的值

value 里的值是 Base 64 编码来的，您可以网上找 Base 64 解码，或者直接去微软商店下载这个应用：DevToys

解码出来，【UID口口】后面开始就是 ID 账号啦，一直到【口口f】，一般是 BFM 开头，多是 20 个字母，这样就可以把这个 ID 账号告诉朋友啦，朋友登录之前，外面没有显示邮箱账号，登录之后，会员中心里显示的信息有部分是 * 号遮挡，这样可以有效防止截图或录像泄露邮箱账号，从而保护了真实的 E-mail 账号。

警告：

只要有账号密码，就能从联系客服里查看到账号的完整邮箱地址信息，使用 BF ID 账号，仅能起到登录前后的账号保护，并不能保护账号密码一起泄露时，盗号者无法得知真实的 E-mail 账号！！！

本文所展示的 BFM7lc0wi4u8220gt1dv ID 账号，是真实的，但是此账号使用随机密码，并且并没有进行进阶认证，只作文章演示使用！

请您在现实运用中，切勿在公共场所，公开您的账号信息！
