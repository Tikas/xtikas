# 小T同学的常用软件集合

前段时间帮一个朋友远程电脑时，竟然把 WPD 写成 WDP ，导致一直无法下载到此软件，之后好不容易才搜索到这个软件名。

由于并没有谷歌搜索，找起来真麻烦，只因字母顺序记错，就带来这么多的时间浪费去搜索，故，在这撰写一篇文章，列出我常用的软件。

# 系统类

  • PE 系统：
    • PE 系统，不用多说，U 盘必备。以前还用过好几个其他好用的，这里只推荐一个就够了。

    【微 PE 工具箱 v2.2 直接下载】 | 【微 PE 官网】

  • 系统优化：
    • DWS：由于被微软封禁，此工具已经消失，虽然有人接手，但是我更想 Nummer 的 Destroy Windows Spying
    • WPD，目前最好用的此类软件了，每次重装系统后第一时间就是使用 WPD 进行关闭相关遥测。

    【WPD 直接下载】 | 【WPD 官网】

    • Dism++，又是一个非常优秀的软件。

    【Dism++ 下载页面】

  • 运行库
    • VC 运行库可以说是系统必备的运行环境了，由于微软并没有在 windows 里集成，需要用户自己去安装，大多时候，小T同学都是使用 All in one 直接安装。

    【Github 微软运行库 AIO 下载页面】

    • DirectX 9：实话，DX 9 存在的意义不大，现在多是 Win 10+ 的操作系统，本身已经自带了 DX 11，但是，如果您的电脑是 Win 7 系统，那么这货就非常重要了。

    【DirectX 9 直接下载（不知道是否长期有效）】 | 【DirectX 9 官网】

    • .NET 6.0 运行库：其实，如果不是我常用的另一个软件需要它，或许 C# 的 6.0 运行库也不是我所常用的。

    【.NET 6.0 运行库直接下载】 | 【.NET 新版运行库官网页面】 | 【.NET 旧版运行库官网页面】

# 输入法

  • Rime-极点五笔86版，由于一直使用五笔作为主输入法，以前一直用着精灵五笔，可惜传闻精灵五笔新版本有后门，当时也确实是去看过官网有新版本放出，由于输入法只要没什么问题，基本就不会去升级，导致错过了当时新版本的下载，现在官方已经删除新版本的精灵五笔，这是侧面证实真的有后门吗？五笔输入法多而杂，每个人的习惯不一样，适合我的，不一定适合大家，请大家客观看待。

    【Rime-极点五笔86版】

  • 搜 G 拼音输入法，这个是方便朋友们的，就不提供链接了。

    链接不见了

# 浏览器

由于我离不开谷歌搜索，一直使用 Google Chrome 谷歌浏览器，虽然也喜欢开源的火狐浏览器，只是火狐浏览器的打开速度总感觉没有谷歌浏览器快。

这里介绍我一些常用的扩展插件：

  • Auto Loop，一代节奏一代神，一代 Loop 一代米，虾米，曾经的常驻之地，现已经消失在历史长河。

    【Auto Loop 扩展程序】

  • uBlock Origin，一直使用这款插件，虽然后起之秀的 Adguard 是那么优秀，但是在电脑端我更喜欢 uBlock Origin，而移动端使用着 Adguard

    【uBlock Origin 扩展程序】

  • GoFullPage - Full Page Screen Capture，虽然用它的时间越来越少了，依旧是我必安装的一个插件。

    【GoFullPage 扩展程序】

  • LastPass，这个不用多说，其实还有另一个非常优秀的管理工具。

    【LastPass 扩展程序】

  • Tampermonkey BETA，一代脚本一代神，代代脚本代代神，神一般的扩展，如果不是另一个程序的滥用，我也不会转用到此工具，当然，更多使用的是 Beta 版本。

    【Tampermonkey BETA 扩展程序】

  • crxMouse，以前一直使用脚本手势，后来发现有些冲突方面的问题，之后就一直使用此插件了，个人不喜欢超级拖曳，必关此功能！

    【crxMouse 扩展程序】

用的扩展非常之多，随着时间的推移，有些已经淡出我的必备插件里了，当然，还有一些是相关领域专用，就不列举出来了。

# 压缩解压

  • 7z，作为 YYDS ， 7Z 无疑是史诗般存在的，由于界面过于朴实，其实我并没有安装它，然而制作相关软件自解压时，是必备的！

    【7-zip 官网】

  • PeaZip，现在我所使用的压缩解压软件，非常不错，只是在 windows 平台里每次打开时，没有 7-zip 那么快，可能是使用 Lazarus 开发的原因。

    【PeaZip 下载页面】

# 音频影视

  • Foobox，听歌，也是我热爱的，但是，我不怎么在线听歌，更喜欢把音乐下载回来（多是无陨 FLAC 格式），然后使用本地播放器慢慢听。由于 Foobar 的界面过于朴实，我更喜欢使用 Foobox 作为我的音乐播放器。

    【Foobox 下载页面】

  • HandBrake，由于非常喜欢视频制作，转码的工具当然必不可少，此款工具目前是我最为喜欢并且常用的，使用它，您需要安装 .NET 6.0 运行库

    【HandBrake 官网】 | 【HandBrake Github】

而视频播放就比较尴尬，因为使用的是绿色版本，加上 【LAV】 解码播放，在这就不列出了。

当然，视频制作类的是 Adobe 系列。

# 图片相关

  • Honeyview 蜂蜜浏览器，作为经常和图片打交道的我，怎么可能少了看图软件呢？而蜂蜜浏览器就是我目前主用的一款看图软件，在设置浏览选项里，不显示其它控件，谁用谁爽，XDD

    【Honeyview 官网】

  • 佳能 DPP，作为一个单反的业余爱好者，佳能的 Digital Photo Professional 软件，是我电脑里必备的，由于这货经常更新，如果您想要下载到最新版本，还需要自己去找，非常不好友。

    【DPP 4.16.0 for Windows】 | 【佳能官方写的查找最新版本 DPP 教程】 | 网上找的序列号：158027002527

  • FastPictureViewer Codec Pack Pro 3.8.0.96，作为一个经常使用 PS 和 AI 的人，显示缩略图也是必要的，这个就不提供链接了。
  • DR 4.5，以前用过很多 PS 相关插件，然而，现在只会保留一个，那就是 DR 4.5 国人汉化版本，因为我已经离不开里面的 统一肤色 这个功能了

# 实用软件

实用的软件比较多，多数是以绿色版本为主，我不怎么喜欢安装太多会驻留后台的程序，加上很多软件都是搞小动作，这类的软件全是以绿色版本为主，以下是列出一些常用的软件，没列出就是没用或者使用绿色版本。

  • ToDesk，目前唯一使用的远程软件了，也只用它，后续有没有更好的替代，到时再说吧！

    【ToDesk 官网】

  • Listary，没有了它，就感觉我的电脑没有了灵魂，虽然作者跑路了，但是同类软件里，还是它最为适合我，我使用的是第 5 版本，最新的测试版本感觉并不好用。由于作者跑路了，建议不要购买 pro 版本，不然，你无法收到 key。

    【Listary 官网】

  • BvSshClient，SSH 里有着许多的好评软件，在 windows 里，我更喜欢这款软件，非常不错！后续有机会找个开源的来取代。

    【BvSshClient 官网】

  • Snipaste，同 Listary 一样，如果没有了 Snipaste，总感觉我的电脑没了灵魂。

    【Snipaste 官网】
