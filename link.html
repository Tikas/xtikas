# 一、Link 传授知识点

Link 传授，可以理解为每个职业都有一个特有的技能属性，传授给其他角色，让这个角色，也拥有这个特有的技能，在使用上，有以下要求：

  • 1、每个角色最多只能接收 12 种 Link 传授（不包括自身职业，有些种类的 Link 传授 可以多个职业一起传授）

  • 2、接收 Link 传授 的角色，不能重复接收同一职业的 Link 传授

    比如，我想 12 个 Link 传授 都只接收阴阳师的传授以达到增加 +120% 伤害。这是行不通的！

  • 3、Link 传授 的技能等级受其传授角色的等级要求，几乎所有职业在 70级 时解锁 1级 Link传授，120级 时解锁 2级 Link 传授，210级 时解锁 3级 Link 传授（3级 Link 传授 仅 GMS 国际服部分职业提供）

    神之子比较特殊，需要完成所有剧情，才能解锁最高级 Link 传授（5级）

  • 4、每个角色的 Link 传授 每天最多传授 两次 给其他职业

    如果您想当天同职业的 Link 传授 大于两次以上传授给其他职业，您可以多练一个这个职业的角色，以达到 2次/天 x 2个同职业传授 = 4次同职业传授/每天，同理，更多就再多准备一个，韩服已经更新每个角色每天可以传授十次，每天前四次免费，后六次需要游戏币。

  • 5、部分职业群的 Link 传授 能叠加使用，只占一个 Link 传授 位置，但是不能接收多个同一职业进行叠加

    比如：冒险家的三位魔法师，叠加后的 Link 传授 有 6级，只能接收火毒魔导师、冰雷魔导师、主教这三个一起进行叠加，不能接收三个主教的传授来一起进行叠加

  • 6、只有接收传授的角色才能享受到传授加成，如果传授的角色被另一个角色接收传授，那么，之前的角色将不会享受到传授加成。

接下来可以查看各职业的 Link 传授属性，也可以直接查看文章尾部的 Link/联盟 图表

# 二、Link 传授搭配选择

这里推荐的 Link 传授，只是符合比较通用的情景，由于每个职业自身特色不一样，如需要对应职业的推荐方案，可以查阅本站对应职业的解读篇。

Link 传授 并不同于 联盟角色卡 那样作用于整个区服内所有角色，它仅能对接收传授的角色起作用。

一个角色需要使用到 Link 传授，无非是用来提升角色实力，而需要用到情景可以说只有两种情景：

  • 1、练等升级
  • 2、挑战首领 BOSS

在这，我们就为这两种情景搭配比较通用的推荐方案：

# 2.1、练等升级的 Link 选择方案

从 【新手入门篇】：前期先练的职业推荐 此文的 高手篇 可以看出练等升级，最为注重的两个要点：

  • 1、提高经验加成
  • 2、提高秒怪效率

为此，练等升级，我们首选的传授加成就应该是这两点，尝试挑选本文前面有那些 Link 传授 是加成这两点，结合一下 小T同学 推荐的方案，看看有什么差异之处：

练等升级的 12 个 Link 传授技能推荐方案

提高经验加成的 Link 只有三个，为高效率练级，毋庸置疑，应该满上。

实战运用时，如果您只想站在地图固定一个地方，不想频繁走动，以轻松的姿势练等升级，那么，狂狼勇士的增加连击宝珠经验就没那么实用了，这情况下，如果伤害不够，可以适当增加伤害的 Link。

接下来，我们需要再分析如何提高伤害来增加秒怪效率：

练等升级，击杀多数是普通怪物，为此，加成多作用于首领的传授就不适合了（如：增加无视、BOSS 首领伤害。）

再加上 爆击伤害 受爆击率影响，如果 爆击率 不能达到较高的理想值（如 80%-100%），增加爆击伤害提升并不理想；当然，如果能传授爆击率以达到理想值，还是可以选择 爆击率 和 爆击伤害。

提高主/副属性和攻击力，也是能提高伤害，加成还是不错的，选上。

还有一些对异常状态造成额外伤害，如果本身职业有自带异常状态（如有些能对敌人附带中毒等 DEBUFF 状态的技能），就非常适合选择这些传授了（因为练等升级，多数处于秒怪状态，不建议选择此类加成！）。

说了那么多，还有很重要的一点：如果练等升级的职业是在推荐选择的 Link 传授 里，那么，可以省下这个职业 Link 传授 占位（职业自带 Link 加成，叠加的不能省，因为其他传授职业还是要占用传授位置）。

以上说的这些，如果传授技能本身有冷却要求，不能达到常态保持加成的，效果会大打折扣，此类应该放在最后选择里。

  • 经验加成：
    • ✅1、精灵游侠：增加怪物狩猎经验
    • ✅2、龙魔导士：增加符文经验 BUFF 持续时间
    • 备：狂狼勇士：增加连击宝珠经验（轻松练等不频繁走动的职业可以选择其他传授代替）
  • 伤害加成：
    • ✅3、菈菈：增加伤害，杀怪再增加对普通怪物伤害
    • ✅4、阴阳师：增加伤害
    • ✅5、恶魔复仇者：增加伤害
    • ✅6、亚克：保持战斗增加伤害
    • 备：伊利恩：移动一定距离增加伤害（如果职业本身多数站着清怪，或者使用轻松练等不频繁走动的，可以选择其他传授代替）
    • 备：虎影：对满血敌人增加伤害（传授后能秒怪就是神技，不能秒请选择其他传授替代）
  • 属性、攻击力加成：
    • ✅7、杰诺：增加全属性
    • ✅8、冒险家三个海盗叠加效果：加全属性、HP、MP，减伤
    • 备：剑豪：增加全属性和攻击力（后期 +15 攻击加成并不理想，好在还有 25 全属加成，如果没有别的加成高，建议更换其他传授）
    • 备：皇家骑士团叠加效果：加攻击加抗性（如果是后期 25 点攻击加成并不理想，建议更换其他传授）
  • 爆击伤害：
    • 备：凯内西斯：增加爆击伤害（爆击率能达到理想值时可以选择，如 90-100%）
  • 爆击率
    • 备：幻影侠盗：增加爆击率（不宜过多，传授一个，要能拯救爆击率到理想值时，如 90-100%，可以选择传授）

以上可以直接选择的，已经标注编号，占用 8 个 Link 传授。

而 7 个备选如果能符合条件，选择优先级是：经验 > 伤害 > 爆击伤害 > 爆击率 or 属性、攻击

其实增加 爆击率 和 属性、攻击 这两个并不好对比，建议您在实战中择优选择（因为后期多是满爆击率，而前期爆击率并不高）。

如果加上备选的还不够 12 个 Link 传授，那么，您可以挑选其他增伤的传授，如：等级压制伤害，异常状态伤害（请考虑秒怪情况下是否适合）。

注：前期多指 200 等级之前，后期多指 200 等级之后。

# 2.2、挑战首领 BOSS 的 Link 选择方案

为了降低进阶首领 BOSS 的挑战难度，这时候，无视敌人防御、BOSS 伤害、满爆击率、爆击伤害，这些加成就非常重要，而经验加成等其他加成就不需要了。

尝试挑选本文前面的 Link 传授，结合一下 小T同学 推荐的方案，看看有什么差异之处：

挑战首领 BOSS 的 12 个 Link 传授技能推荐方案

在 Link 传授 的技能加成里，并没有增加最终伤害的传授加成，不考虑具体装备加成、职业本身加成、并且默认爆击率是 100% ，无视敌人防御不低的情况下，Link 传授 选择优先级是：

BOSS 首领伤害 > 伤害 > 爆击伤害 > 无视敌人防御 or 主/副属性

警告：此优先级仅适合 Link 传授，并且应该满足前面所说相关情况。请不要以偏概全套用此优先级！！！

爆击伤害 如果在满爆击率的情况下，是直接提升的。无视敌人防御 由于计算复杂，在大于 95% 之后收益并不高，建议您保持无视敌人防御在 93% 以上，这两个具体那个加成高，还需要您结合自身实际情况进行伤害提升计算。

    • 次选：伊利恩：移动一定距离增加伤害（实战中可能并不能享受最大加成）
    • 次选：凯殷：杀怪或攻击首领增加伤害（激活后加 17% 的伤害，持续 20 秒，配合爆发非常不错）
    • 次备：菈菈：增加伤害，杀怪再增加对普通怪物伤害（才加 5% 的伤害，如果有位置可以选择）
    • 特殊：对附有异常状态的敌人增加伤害（最好一起传授，如果职业本身自带技能附加异常状态被动，效果非常好）：
      • 冒险家魔法师叠加效果：攻击时有概率增加伤害和无视（需要触发后才加成）
      • 冒险家盗贼叠加效果：对附有异常状态的敌人增加伤害（有异常状态直接加成）
      • 卡蒂娜：增加异常伤害、等级压制伤害（有异常状态直接加成）
  • 爆击伤害：
    • ✅6、凯内西斯：增加爆击伤害
  • 无视敌人防御：
    • 夜光：增加无视敌人防御
    • 虎影：增加无视，对满血敌人增加伤害
    • 神之子：减伤加无视
  • 主/副属性：
    • ✅7、杰诺：增加全属性
    • 次备：冒险家海盗叠加效果：加全属性、HP、MP，减伤
    • 次备：剑豪：增加全属性和攻击力（相对来说，加成并不多，有位置时可以选择）
  • 复活后无敌：
    • 次选：末日反抗军叠加效果：增加复活后无敌时间（如果需要拿命去拼，就带上这个传授吧！技巧：可以在复活时替换此传授）
  • 机率不死：
    • 次选：隐月：受到致命攻击时机率不死（有空余位置，不如带上此神技，感受人品带来的魅力）

这里推荐的方案，是假设您的角色已经满爆击，并且对无视需求不大时，进行推荐的，如果您不能满足这两点，请优先满足：

无视敌人防御 > 满爆击率

无视提升需要结合自身实力进行伤害提升计算，受每个人的实力不同，无法给出具体答案，建议您无视敌人防御最好保持在 93% 以上

以上可以直接选择的，已经标注编号，占用 7 个 Link 传授。

次选里可以选择符合自己需求的，具体选择功能性还是实用性，这就因人而异了。那怕同样是实用性，其 BOSS 首领伤害 是否大于 伤害 加成，都不相同，这需要结合自身实力进行伤害提升计算，才能得知。

而特殊的就非常适合那些本自能技能附带异常状态的职业选择，当然，选择特殊的，您需要最少占用两个传授位置（选择直接加成异常状态伤害的），而最好是三个全选，这样就需要占用三个 Link 传授 位置了。

而次备，就是您在次选里面不能找出符合自己需要的，可以从次备里找找。
