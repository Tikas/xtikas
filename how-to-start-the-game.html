🎮 新枫之谷的启动方法

 大约 9 分钟

由于新枫之谷使用了繁体编码，如果您在简体操作系统直接运行游戏，就会提示：

This client can be executed only in Traditonal Chinese code-page

不用担心，我们只需要知晓三种启动游戏的方法，就能体验游戏了。

# 三种启动方法之间的比较

虽然本文一直在强调目前主流有三种启动游戏的方法，实际情况中，其实只需要两种就可以，另一种只用来备用。

详细可以通过下面表格对比得出：

对比项 原生态网页启动 使用 LE 启动 第三方缤放登录器启动
不用修改系统语言环境 ✘ ✔ ✔
不需要 .Net 运行库支持 ✔ ✘ ✘
记住密码风险性 IE易泄露密码，风险大 不支持账号密码 相对安全
多账号切换 ✘ 不支持账号密码 ✔
一步到位进入游戏 ✔ 仅启动游戏，不能登入 ✔
星级得分 💛💛🤍🤍🤍 💛🤍🤍🤍🤍 💛💛💛💛🤍

由于我们大多注册的都是 港号，港号使用了较旧的技术，仅支持古老的 IE 浏览器，从上图对比可以看出：

原生太网页启动：适合单个账号，并不需要记住密码的用户使用，仅需要修改一下系统的语言环境（非常容易修改）就可以做到直接启动游戏并正常畅玩。

使用 LE 启动：LE 是 Locale Emulator 的缩写，它能让您的电脑不管处于什么语言环境，都能运行指定语言环境的程序，神器！但是它本身不进行相关账号密码管理，因此，很多单机游戏会用到它。此方法仅仅只是支持游戏程序能正常启动，由于本身不支持账号密码，导致无法登录游戏，眼看着登录界面，却什么也做不了。此方法多用来备用。

第三方缤放登录器启动：第三方登录器本身是采用了 LE 来启动游戏，让您可以在简体操作系统里直接启动游戏。并且，由于本身集成了账号管理，可以非常方便进行多账号登录操作。唯一可惜的是：它使用 .Net 开发，配置低的电脑再运行 .Net 开发的程序会占用掉大量的系统资源，并且，如果操作系统不是 Win 10 以上的话，会容易出现一些特别情况。

现在大家会对如何选择适合自己的启动游戏方法有一定的了解，接下来，分析一下他们之间的实现方法，请往下看。

# 方法一：原生态网页启动

要想使用网页启动游戏，你需要把电脑的区域设置为繁体台湾。

打开：控制面板，在控制面板里找到区域。如果你的是 “类别” 显示，你应该先找：时钟和区域，再在里面找到 “区域”

在区域设置的 “管理” 非 unicode 程序的语言里，点击 “更改系统区域设置”，选择设置如下图：

设置繁体区域之后，建议不要使用老版本的第三方缤放登录器来启动游戏啦，不然容易报：C00700C1 错误。

这样就可以在简体操作系统直接启动新枫之谷啦。

# 方法二：使用 LE 启动

使用 LE 来启动游戏，可以比较好解决 第三方缤放登录器 3.x 版本 卡死的问题。

启动游戏之后，可以配合使用第三方缤放登录器获取密码来进入游戏。

这是一个非常不错的搭配方法。

感兴趣的小伙伴们，可以网上搜索查相关文章。

# 方法三：第三方缤放登录器启动

使用 “第三方缤放登录器” 可以很好解决 “This client can be executed only in Traditonal Chinese code-page” 这类问题，因为 “缤放” 已经引用了 Locale Emulator 模块，简体系统，一样玩新枫之谷！

由于涉及到资金安全，建议大家下载回来的第三方缤放登录器，先进行安全检测。

在线查杀：中文 virscan (opens new window) | 国际英文 virustotal (opens new window)，结果仅供参考

由于 “第三方缤放登录器” 使用 .Net 开发，如果您是 Win 7 系统，您还需要更高版本的 .Net 运行库，以解决缤放登录器未响应或已停止工作等问题。

前往：微软 .NET Framework 下载页面 (opens new window)（这里以新版本 4.8 版本为例子，选择对应版本之后，点 Download .NET Framework 4.8 Runtime 下载）

如果已经使用最新版本的 .Net 运行库，依旧提示错误或者未响应等情况，您需要清理 .Net 运行库的缓存，具体可以使用 Dism++ 工具来清理。前往：Dism++ 官方 Github 下载 (opens new window)

在设定界面指定一下新枫之谷的游戏路径，设定按钮在 “缤放” 的右上角那一排可以找到。

设置界面如下：

其中选中 “传统登入模式”，就会有 “获取密码”，这样就能获取动态登入游戏了。

提示

如果您是新人，建议不要选中 “传统登入模式”，这样点启动游戏就会直接登录您的游戏账号。

如果没有勾选获取密码旁边的 “自动输入”，那么您需要手动复制游戏账号及动态密码。

游戏账号需要在 第三方缤放登录器 软件界面里双击自己的账号名称，就会复制一段类似乱码一样的字符串，这就是游戏的账号；密码需要点 获取密码 才会出现。

警告

千万不要在游戏登录界面里输入注册的 Beanfun 账号和密码来登录游戏！

新枫之谷（台服冒险岛）使用的是 Beanfun 账号获取游戏账号的动态密码进行登录游戏的！

如果不知晓怎么查看自己的游戏账号和密码，可以看下面的：“打开查看：启动游戏会提示：未选择启动游戏的账号”

打开查看：启动游戏会提示：未选择启动游戏的账号

如果你的账号是刚刚注册的，登录缤放之后，是无法获取密码的，你还需要进行下列操作：

1.在 第三方缤放登录器 主界面右下角，点击 “新增账号”

2.在 “使用者名称” 里输入注册账号时填写的昵称（其实可以随便填的，但是发生过 BUG，这个 BUG 应该不会再出现了。）

如果您无法正常新增账号，您需要使用 IE 浏览器，在浏览器里网页在线增加！

3.如果你的 第三方缤放登录器 底下没有获取密码按钮，那么你需要在 “设定” 里选中 “传统登入模式”

4.新版本 第三方缤放登录器 默认是隐藏了账号，你可以在账户名称里双击复制你的账号。（如果选中自动输入，获取密码后，第三方缤放登录器会自动帮你输入账号和密码）

# 没看这一步的人都哭晕了 重要

第一次提示的密码为二级密码，非常重要！

创建角色、进入仓库、现金道具交易等都需要它，尽量简单易记就行，别太复杂！

如果忘记二级密码，您只能邮件联系客户取得帮助，非常麻烦！

游戏角色名一定要包含中文！

创建游戏角色名请一定要包含中文！不然容易锁号！

别起个全英文，或者英文加数字的角色名，一天一锁别来哭诉！

每周三维护

每周三会进入维护，一般是早 8点 到中午 14点，大维护时期具体时间请看官网公告。维护期间是无法进入游戏的！

刚刚注册的账号别急着进入游戏

如果账号是刚刚注册，别急着进游戏，先放松几分钟，不然，会提示下面错误：

等几分钟后（这个过程可能是需要同步港号的相关数据，有时需要隔一两天），就可以进游戏啦！

如果你等一段时间还是提示这个，请确认你的账号是否进行了两次进阶认证！

好像从 2020年 3月份 开始，就提示为 “请确认您的 Gash 账号与游戏账号是否为正常状态”
