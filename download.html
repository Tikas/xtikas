# 一、当前最新游戏版本

当前游戏最新版本为: v242 版本（同步更新于: 2022/04/04）

TIP

如果当前版本后面有带 .X 那么，后面这个数字就是小版本号，您需要下载当前大版本之后，还需要下载小版本进行更新，才能正常进入游戏。

如果后面没带小版本号，如：v235 这个大版本，其实默认是自带一个小版本号：v235.1 ，您可以通过右键 MapleStory 启动程序查看版本信息。

# 安装 v242 大版本游戏

由于新枫之谷使用了繁体编码，在简体系统中运行会显示乱码，但是，并不影响整个游戏的安装

运行解压出来的安装主程序即可进行安装 (很容易猜出下一步按钮)

安装目录一定不能有中文

如果游戏的目录路径带有中文，您将无法启动游戏

安装过程出现异常错误提示（乱码显示）

如果您安装过程中一直会提示相关对话框，不能顺利进行安装，您下载回来的游戏文档可能并没有下全。

请检查一下分割包 bin 的文件大小是否都一致：

一般前面的几个 bin 文件，大小都是一样的，只有最后一个 bin 文件才会小些。

您可以尝试修复一下下载回来的 zip 文件, 如果修复后还是会提示, 建议您重新下载游戏文件

乱码显示是由于程序使用繁体编码，如果您的系统没有更改区域语言，就会显示乱码

# 安装新枫之谷小版本更新档文件

小版本更新档文件多数是 7z 自解压文档来的, 您可以很直观通过文件的图标识别它

建议右键以 管理员身份运行 小版本更新包程序

TIP

如果当前版本并没有小版本更新档，请您跳过这一步。

如果有多个小版本更新档，请先按小版本数字从小到大排序，顺序安装。

# 3.1 运行新枫之谷小版本更新档程序

从上图可以看出，手动更新包运行时，获取的目录路径是当前的。

可以把手动更新包拷贝到游戏目录内运行的哦！

为了减少游戏目录所占用空间过大，推荐把更新包存放在其他地方。

# 3.2 指定新枫之谷游戏目录

点击 ... 按钮，会弹出浏览文件夹对话框，指定到新枫之谷游戏的目录路径

如下图所示（这是小T同学自己的游戏目录安装地方，并不是您的电脑游戏所在目录）

MapleStory 目录下有：BlackCipher，blob_storage，GPUCache，locales，swiftshader 等目录

那么这个 MapleStory 就是游戏目录路径啦！

本来不想多说这一步的，实在是太多小伙伴游戏目录类似这样的：.../MapleStory/MapleStory/MapleStory.exe,然后他们分不出到底那个 MapleStory 才是正确的游戏目录

# 3.3 Extract 解压覆盖

确保游戏目录没有错误，就开始点 Extract 按钮，解压覆盖吧！

提示下图所示，就是解压到正确的游戏目录了，点：Yes to All 吧！

# 四、旧版本升级到最新版本

如果您电脑里之前有新枫之谷，那么您可以前往游戏目录，右键游戏的启动程序：MapleStory.exe，选择属性，查看一下 详细信息 里的产品版本，可以得知当前旧游戏的版本信息。

再查看一下是否有对应的升级文件，如果有，就可以下载升级文件更新到最新版本。

当然，更新到大版本之后，还是要检测一下当前大版本是否有出小版本更新，具体可以参考本文前面的内容：四、下载小版本更新档文件
