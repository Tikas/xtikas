# 使用

由于第三方缤放登录器采用 C# 开发，如果您的电脑里没有 .NET Framework Runtime ，您会无法使用此软件。

如果您的 .NET Framework Runtime 版本过低，也会无法使用。

推荐使用 v4.8 + 版本，并不是说一定要安装。

TIP

如果您的电脑能正常运行程序，请跳过这一步!

如果您是 Windows 7 系统，使用 4.8 版本依旧会出现相关问题，请您换 v4.6.2 版本。

前往：微软 .NET Framework 下载页面（这里以新版本 4.8 版本为例子，选择对应版本之后，点 Download .NET Framework 4.8 Runtime 下载）

如果已经使用较新版本的 .Net 运行库，依旧提示错误或者未响应等情况，您需要清理 .Net 运行库的缓存，具体可以使用 Dism++ 工具来清理。前往：Dism++ 官方 Github 下载

不出意外，您能看到登录器首次运行的区域选择提示！

DANGER

请不要选中 自动登录，不然，很容易出现卡死、启动无界面、闪退、提示模块等等错误！

# 内置实用的小工具

登录缤放之后，在主界面，可以找到一个 “工具” 按钮，里面有一些很实用的小工具哦！

其中有最实用的 装备卷轴星力计算器，我们一起来看看怎么使用吧！

使用方法是：选择对应的装备部位，对应的等级，在 A 和 B 里填写装备自身的属性，就可以啦。

其中：

  • A、装备当前星力强化等级（就是多少星的装备啦）
  • B、是装备原本白色的属性值，这个一定要填写哦，不然像武器类他的攻击加成是按装备自身多少攻击来算的。
  • C、这是星火属性值，写上去，如果最终上卷正确，就能在左边的总值和装备对得上，不填写这个就会少星火属性。

也许最终加上卷的值，装备的属性会比计算器的多一些攻击，这是因为天然白板装时，就有浮动的属性加成。

注：如果是使用荣耀卷轴，数值多数会不准，一般都是大于黑卷，㘬值有小数。

除了装备卷轴星力计算器，新版本还有一个新的 完美核心计算器 也值得去使用哦！

# 🆘 新版本 64 位游戏相关问题

# A:) 提示：以非繁体语系系统启动游戏的方式不支援64-Bit游戏

由于 MapleStory 全面启用 64 位应用程序，如果您的登录器版本低于 4.3 版本，将无法在简体系统正常使用登录器启动游戏。

旧版本使用的是 LE 模块来进行转区，由于 LE 模块不支持 64 位应用程序。为了支持 64 位的 MapleStory，新版本使用了 InWILL 开发的 Locale Remulator 进行启动游戏，新版缤放登录器已经集成了，您不需要额外去下载 LR。

解决方案

请您去缤放登录器的 Github 下载页面，下载最新版本。

# B:) 提示：RunDLL 启动 LRHookx64.dll 时出现问题

如果您出现了提示此 RunDLL 错误，找不到指定的模块，并且缤放登录器同级目录下有以下两个 DLL 文件：

  • LRHookx64.dll
  • LRProc.dll

那么，这情况是由于 Microsoft Visual C++ Runtime 导致的！

由于 InWILL 作者使用 VS 2019 开发和编译 Locale Remulator，而缤放登录器的作者也是引用 LR 来启动游戏，错误的 VC 运行库版本，导致了这个问题的发生。

解决方案

您可以前往下面链接下载最新的运行库，安装完之后，就可以正常使用第三方缤放登录器启动游戏了。

  • 1、下载 VC x86 运行库 ｜ 下载 VC x64 运行库 两个都安装之后就可以正常使用了。
  • 2、前往 Github 下载所有的微软运行库，这个是包含了全部的 VC 运行库，可以同时解决依赖其他版本的软件正常使用。

  • 1、运行库、 .Net Framework 出现错误
  • 2、系统相关核心出现错误

建议先尝试第一个解决方法。

# D:) 复制的文本在游戏里显示乱码

这个问题是由于 Locale Remulator 并未做剪贴板内容转码处理导致的。

解决方案

请下载 v4.4+ 版本。

# E:) RunDLL 启动 LRHookx32.dll 时出现问题

如果提示找不到指定的模块 LRHookx32.dll，那么，您应该卸载相关数字安全软件，如果喜欢国产的安全软件，建议使用火绒配合微软自带的杀毒来提高安全。

如果当前游戏出了小版本更新，您并没有进行更新，也会出现此提示！

因为游戏出了小版本，会运行在线更新程序，这个程序是 32 位的应用程序，就会导致第三方缤放登录器启动时提示 LRHookx32.dll 模块。

但是，默认情况下是禁止自动更新的（原因可以查阅本文相关内容），如此便会出现此错误。

请手动更新游戏之后，play 窗口之后就不会去启用更新程序，如此就不需要 LRHookx32.dll 模块了。

解决方案

请更新游戏为最新版本。

并且保持 v4.4+ 版本的第三方缤放登录器，如果还不能解决这个问题，请您卸载相关数字安全软件。

# F:) 释放区域模拟插件错误，请重新启动电脑后再次尝试

如果您是港号，并且选中了自动登录，那么，您将会出现这种情况，请不要选中自动登录！！！

解决方案

  • 1、打开运行窗口（按 Win + R 键）
  • 2、在运行窗口里输入：%APPDATA%（注：复制时请包括前后两个 % 号）
  • 3、在弹出的资源管理器里找到 Beanfun 目录，如：AppData\Roaming\Beanfun
  • 4、删除里面的 Config.xml 文件（注：另一个是存放账号信息，删了就没账号记录了，只有此文件才是缤放登录器的配置文件）
  • 5、再次启动，请别再选中自动登录！

之后建议把第三方缤放登录器更新到最新版本。

# 1️⃣ 缤放登录器启动类问题

# A:) 启动第三方缤放登录器提示已停止工作

由于第三方缤放登录器使用 C# 语言开发，如果您的电脑没有 .NET 运行库或者运行库版本比较旧，就会出现这情况。

解决方案

解决方法可以参考本文的第二章节：使用

# B:) 第一次正常，之后每次启动缤放都卡死闪退

请不要勾选 自动登入 ，由于 @Pungin 对 3.x 之后的版本进行登录重构，导致自动登入非常容易发生缤放登录器卡死、闪退等现象。

如果您已经勾选 自动登入 并发生了卡死闪退等现象，您可以在登录中界面狂点 取消，或者参考前面的清除本地登录器的配置文件。

# C:) 系统错误：无法创建新的堆栈防护页面

这个问题是由于 NET 程序集缓存溢出导致的，其发生问题多是以下两个原因：

  • C 盘 NET 程序集缓存目录存放太多老旧的缓存导致溢出
  • 内存条的内存分配出问题，也许是系统运行太久导致内存溢出

解决方案

您可以使用 dism++ 的 常用工具 - 空间回收 - 缓存文件 - NET程序集缓存（by 大胖子技术员） 进行清理，清理完之后，重启电脑，就可以让电脑的内存条断电，从而达到清空内存缓存。

简单理解：使用 dism++ 清理 NET程序集缓存 之后，重启电脑。

您可以从 这里 获取 dism++

如果您是 Windows 7 系统，请下载 .NET 4.8 版本的运行库。

如果你的缤放登录器是 2.9.9 版本之前的，那么提示无法登录账号，就是没有安装橘子官方的 beanfun 平台程序导致的

如果是 3.0 版本之后的，那么你可以看看是否网络问题导致的，有时 网络故障 就无法登录了，请换加速器看看。

# C:) 提示:无法加载进行反序列化所需的类型 beanfun records

从 3.0.1 版本开始，Pungin 作者重新更新了本地记录文档，旧版解析新版本的本地记录文档就会提示 “无法进行反序列化”

解决方案

请重新下载新版本的第三方缤放登录器就可以解决这个问题。

不建议删除账号记录文件以启动旧版本为目的的操作

删除缤放账号记录文件方法:

按 “Win + R” 键打开运行，在运行里输入：%appdata%

在 “AppData\Roaming” 里删除 “Beanfun” 目录就可以。

如非必要，请不要删除此文档，不然，您所记录的所有账号密码将清空！

# D:) 检查到档案数位签章错误，请洽客服人员

前往 beanfun 官方外挂程式下载页面，下载并安装 beanfun 官方外挂程式，即可。

# E:) 台号一直提示登入失败，未找到 Eventvalidation 讯息，请检查网络连接

由于台号的网页登入页面改版了，您需要下载新版本的第三方缤放登录器，才能正常登入。

解决方案

如果您的缤放登录器版本低于 v3.1.5 版本，您是无法登入台号，请下载新版本。

# F:) 请点击我不是机器人

这是由于加速器的同一个节点太多人使用，导致需要进行人机验证

解决方案

需要您进行更换节点试试，或者更换其他加速器。

# G:) 港号登入失败，账号或密码错误。

如果您使用记住密码，或者能确定密码是正确的，此时登录账号，还会提示：

登入失败，账号或密码错误。（LoginNoProcessLoginV2JSON）

这应该是您的加速器并没能加速香港 beanfun 元件程序，导致的

解决方案

请您更换一个节点，或者换别的加速器。

有的加速器由于没有加速到香港 beanfun 后台服务程序，导致无法登录，此时就会提示此错误。您可以停用加速器登录缤放登录器后，再启用加速器，此时不影响动态密码获取。 IP 异常也会提示这个，请换稳定的 IP 节点，等待 5 分钟之后尝试。

# H:) 点击登录就缤放消失不见

这种情况多发生在港号，是由于频繁更换 IP 导致异常，只需要换一个稳定的节点，耐心等待 5 分钟之后就可以再次尝试了。

这情况好像还会提示账号密码错误。

# A:) 指定游戏目录，才能用缤放启动游戏

如果您是通过第三方缤放登录器来启动游戏，您需要先为 设定 里指定一下游戏目录。

设定 可以从主界面的右上角里找到按钮。

其中选中 “传统登入模式”，就会有 “获取密码”，这样就能获取动态登入游戏了。

TIP

如果您是新人，建议不要选中 “传统登入模式”，这样点启动游戏就会直接登录您的游戏账号。

WARNING

如果您指定的是桌面游戏的快捷方式，此时启动游戏将会提示：

Themida - http://www.oreans.com

Cannot fine 'CrashReporter.dll'.please,re-install this application

请您修改第三方缤登录器的游戏路径为安装的目录。

# B:) 启动游戏后黑屏并闪退

如果您启动游戏之后，是全屏显示游戏，并且是刚进入全屏时就闪退，这时，您应该考虑是显卡驱动问题。

解决方案

您可以尝试以下两个方法解决此问题：

  • 1、刚进入全屏是，快速按一次组合键：Alt + Enter(回车键)，切换到窗口模式进行游戏。
  • 2、尝试不同版本的显卡驱动，直到测试出支持全屏游戏的驱动（过程比较麻烦）

建议优先使用第一种方法进行游戏，后续有时间再来解决全屏游戏闪退的问题。

很多加速器加速效果并不理想，也会导致进入游戏界面时闪退！
如果是此情况，建议选择白嫖的免费加速器：2个n点com（由于没有收到此加速器的推广费用，不知道会不会给我打钱）

# C:) 缤放启动游戏后，右下角只提示 NGS，就没了

启动游戏之后，跳过了 play 界面，并且右下角的 NGS 安全图标也提示了，之后，并没有出现游戏界面

这情况往往是您的游戏安装目录路径带有中文导致的！

请把游戏的父级目录里中文改成非中文，或把新枫之谷（台服冒险岛），拷贝到别的地方。

也许您不能理解 路径 这一词，没事，我来举个例子：

  • ✅ C:\Program Files\MapleStory\MapleStory.exe
  • ❌ C:\Program Files\新枫之谷\MapleStory.exe

上面两个路径，前面一个为正确示例，后者带有中文，就是错误的路径示例。

当然，有时候 NGS 进程卡住了，也会出现这种情况，如果您的游戏路径并有没中文，请考虑结束 NGS 进程，以方便正常启动游戏。（结束 NGS 进程，可能需要再次启动游戏）

另外，还有一种情况：

如果您的游戏目录存在旧版本的文件，当您重新安装最新版本到旧游戏目录时，会把旧版本的 NGS 文件保留部分，这情况也会出现 NGS 提示后不出现游戏界面。

简单解决办法，清空此目录，重新安装一次游戏即可。（由于并不能确定具体旧版本留下的相关文件，故，在此只列出此笨方法，请见谅！）

# D:) 来自缤放的警告：自动更新可能破坏程序，已被阻止

这是由于第三方缤放登录器默认是阻止自动更新的！

这是由于我们都是使用加速器来玩新枫之谷的，在线更新很容易发生数据丢失（就是丢包现象，平时玩的时候会卡顿）。

出现这个提示，意味着有新的游戏版本，建议去官网查看，或者来本站的下载文章查看。

新枫之谷官网下载页面 | 小T撰写的游戏下载文章

# E:) 启动游戏之后，第三方缤放登录器就卡住

这个现象一般出现在 v3.x 版本，这是当时相关代码没优化好导致的，简单的解决方法就是使用新版本。

当然，您也可以使用 LE 来启动，配合第三方缤放登录器只获取密码来登入游戏，但是这个方法多了一些步骤，相对来说没有那么便捷，不推荐。

# F:) 非繁体语系系统启动游戏失败，提示：C00700C1

有三种原因导致这情况出现：

  • 1、使用了自动更新，导致游戏文件被损坏，请重新安装更新，或者重新安装一次游戏，再或者找朋友另外复制一个游戏主程序过来试试（建议大家别用自动更新）
  • 2、更改了语言区域导致区域异常，也会提示这个，解决办法就是简体语言和繁体语言换一换来尝试看看了
  • 3、被安全软件拦截了 LE 模块，比如使用了 3** 相关安全软件。建议到 msdn itellyou 里下载镜像重装系统！

# G:) 启动游戏提示错误码：C0000135

这可能是第三方缤放登录器的 LE 模式被相关安全软件拦截，或者你的系统本身之前有修改过其他语言区域。

解决办法是修改回简体语言环境，或者改为繁体语言环境。

# H:) 启动失败，请尝试从桌面捷径启动游戏。若您系统为非繁体语係系统，可能是 Locale Emulator 元件不支援您的系统。

这是由于您的缤放目录下，缺少 LocaleEmulator.dll 和 LoaderDll.dll 文件导致的。

这多发生在 v3.x 以下版本，主要是第三方缤放登录器并没有把 LE 模块释放到系统目录，只读取登录器同级目录。

新版本已经不存在这个提示，如果还有，请检查一下是否有安全软件拦截了登录器释放 LE 的 DLL 文件到系统。

# 4️⃣ 游戏账号登入类问题

这里的问题，多数是网络的原因，如果您更换节点不能解决，那么请更换一下加速器，或许多数问题就会得到解决。

# A:) 登入失败，未找到 Eventvalidation 讯息，请检查网络连接

如果出现这情况，请留意一下账号安全情况！

登入状态除非很久，或者更换过节点等操作，正常情况下是：不会执行退出账号行为的！

那么，您刚登录账号不久，再获取密码就提示这个，应该是有人登录过你的账号了！

具体登录记录可以从：会员中心-查询记录-大事纪-beanfun!主账号 查询账号登录情况。

细心的朋友，也会发现前面台号账号登录缤放时也有此提示，如果您不是此情况，请参考前面答案。

TIP

偶尔港号服务器出问题也会出现这个提示（港号的服务器貌似经常出问题，真服了）

# B:) 没有添加账号，启动游戏会提示：您还未选择需要启动游戏的账号

出现此情况一般是以下几种：

  • 1、刚刚注册的账号
  • 2、加速器问题
  • 3、极少见服务器 GG 了导致的现象（其中港号相对来说比较常见）

解决方法

1.在 第三方缤放登录器 主界面右下角，点击 “新增账号”

2.在 “使用者名称” 里输入注册账号时填写的昵称，或者随便填一个。（其实随便填的发生过 BUG，这个 BUG 应该不会再出现了。）

如果您无法新增账号，您需要在浏览器里网页在线增加！（港号请使用 IE 浏览器）

解决方法

可以尝试更换节点，或者使用别的加速器尝试（许多加速器都是永久免费的，由于并没有收到广告费，在此就不列举了）

# C:) 无法获取密码

有多种原因会导致这种情况出现：

  • 1、港号未安装官方 beanfun 外挂程序，或未连接上程序（未连接上请考虑是否有安全软件拦截，或者前往 beanfun 官方外挂程式下载页面，重新安装一下）。
  • 2、加速器导致，或网络问题。有些加速器是进程式加速，而多数是 IP 式加速，可以考虑更换其他加速器尝试。IP 式加速器，还能有效避免锁号问题。
  • 3、特殊情况，如：2020年5月底因大量工作室涌入导致暂时关闭新号密码获取。这是短期的，等这情况过后，就会恢复。

# D:) 获取密码失败，服务账号不存在

之前就有多个萌新深夜去注册账号，无论等待多久都提示：“服务账号不存在”，但是查询账号信息都显示状态正常。

经过多位小伙伴提供的注册时间可以得出：深夜后注册的账号等于空号！这时候您就需要通过 “客服中心” 联系客服，帮忙解决此问题

请大家别在深夜去注册账号。

另外，每次维护时注册的账号也会出现这情况，维护时间多是在星期三，这是由于数据同步不到游戏数据库，这情况就需要等待一天了。

要是等了一天多（2022年03月有小伙伴说要等两三天后才能正常获取，可能数据同步需要更长时间了），还是不行，就需要 联系客服进行账号异常处理
著名服务器卡爆事件也会导致此类情况发生
2020年5月底大量工作室进 TMS，导致服务器卡爆，官方直接把月底几天申请的新号暂停密码取得服务，并且加强 GM 在线排查。这时候申请的账号就会获取不到密码。
后续会不会再次出现这种情况也很难说，好在这种特殊情况只维持几天。

# E:) 锁号误封或者永久封号的账号无法进入游戏

如果账号显示灰色，要么是锁号了，要么是封号了。

请查看原因，可以点 “会员中心” - “查询记录” - “大事件” 里查看。

之前大事件不能查询当天的，最早是昨天的。现在不知道是否可以查询当天的了。

事件显示：帳號鎖定(終止遊戲契約(永久鎖定);0)。

出现这情况，会有两种情况，一种是使用非法程序导致永久封号，如：压键盘、使用鼠标连点、外挂程序、购买工作室游戏币，等等。

如果不是上面所说的情况，那么应该是误封了，请申请解锁。如 2020年3月份 有名的大面积永久误封斗燃猫号，只要申请，说明原因，就会得到解锁。
