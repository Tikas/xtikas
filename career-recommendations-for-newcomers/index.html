# 新枫之谷（冒险岛）前期先练的职业推荐【新手入门篇】

本文并不是推荐大号（主号、本尊）职业，只是介绍新号刚开始需要练那些职业以提升实力。

由于每个人刚玩时的条件不一致，小T同学 就在这里进行了分类：

  • 完全新人，刚刚入门，人称：萌新
  • 怀着零氪来体验游戏
  • 有大佬提携
  • 有自己的游戏了解，不需要别人指点
  • 懂得利用资源，并且自己有时间可以肝，租赁（或购买）相关道具自己练号
  • 我是大佬，练号是痛苦的根源，还是请代练（也可能请的是外挂或洗号）来练吧

以上几点，虽然不全面，但是也可以说囊括了大部分人群了，为了撰写一篇适合大部分新手玩家的《前期上手职业》，本文写的比较直白，希望能给您带来一些收获。

本文以以下两点为核心：

  • 新人前期，能有个良好的游戏体验
  • 高手前期，能有更高效的游戏起步

# 一、特殊时期，直接的创建角色

  • 神之子：
    • 神之子一直以来都是限时提供（多为两年一次限时提供），如果您当前能看到可以创建 神之子 这个职业，请创建一个，放着就行，等后续对游戏比较了解了（当您懂得 link 传授及联盟方阵之后），再来进行神之子的冒险旅程。
  • 查看当前 燃烧 活动，如果活动快结束，请直接创建一个燃烧角色开练。
    • 燃烧活动，是每个大版本都会有的活动，如果当前活动快结束了，不妨创建一个燃烧角色直接冲等级就行了，因为很快就下一版本到来，新的 燃烧活动 出现，旧的 燃烧活动 消失。长话短说：冲就完了。至于选择那个职业，可以接着往下看。

# 二、新人篇

新人刚玩，对游戏的理解并不是很深，为了能提高前期的游戏体验，在不考虑租赁装备等情况下（主要防止被骗被坑），列出了下面几个方案，不妨参考参考：

# 2.1 开局无压力，上手更神速

由于 MapleStory（国服叫冒险岛，台服叫新枫之谷）有着众多职业，每个职业的特色也并不相同，前期尽量选择一些容易成长的职业或者对后期帮助很大的职业。

作为纯新人，请不要直接听信大佬所说的先练三个经验加成的职业，因为您可能还不知道有那些经验加成道具，如果不能在经验拉满加成的情况下，并且又没有相关强力装备加成，这点经验加成并不是很理想。

而空号（新号）去练 精灵游侠（CMS国服又名：双驽精灵），死亡将会是常态，这对游戏体验来说并不好。

为了能早点开启联盟方阵，新号开始建议选择以下职业：

  • 不使用燃烧活动，非常适合开荒的首选职业：
    • 天生自带宠物
      • 幻兽师（CMS国服又名：林之灵）：

      幻兽师虽然被取消了很多成长奖励，依旧是新号最好的起步职业之一。天生自带宠物，拥有强力的清怪效率，所以，请您一定 不要 为幻兽师创建为 燃烧角色

    • 自带回复技能
      • 恶魔复仇者：

      由于恶魔复仇者的头发是白色，又名叫：白毛。是唯一一个主属性是血量的职业。加上其天生带有满回复的技能，可以说是血厚防高还省药水，前期那怕没有宠物也能轻松提升。而其 Link传授 及 联盟加成 也是必备的。

  • 燃烧角色（燃烧的角色，自带宠物）
    • 当前的极限燃烧活动是大斗燃（10-200 级），在活动时间充足的情况下，可以把燃烧角色留给自己想玩大号（本尊、主号）的职业，或者给下面职业：
      • 阴阳师：

      阴阳师是一个非常轻松就能达到 200等级 的职业，其 Link传授 及 联盟加成 同恶魔复仇者一样，同属于必备加成。

    • 当前的燃烧活动是小斗燃（超级燃烧是 10-150 级，燃烧加速器是 10-130 级，建议 Link 传授 角色到 140 等级先停下练等），可以先练一些对后期比较实用的职业：
      • 菈菈（CMS国服又名：啦啦）：

      当菈菈到达 120 等级时，能有 2 级的 Link传授，传授加成为：增加伤害和怪物伤害，多数时间能提升 +16% 的伤害，对前期练号帮助非常大，是最好的起步职业之一。
      如果是 大斗燃（10-200 级的极限燃烧活动），也可以练菈菈，从实用角度来衡量，最好能给阴阳师，毕竟阴阳师的联盟加成更香。

      • 杰诺（CMS国服又名：尖兵）：

      当杰诺到达 120 等级时，能有 2 级的 Link传授 ，传授属性为：全属性 + 10% ，非常不错的提升属性，由于杰诺前期清怪效率并不算高，如果当前的燃烧活动是 小斗燃（10-150 级的超级燃烧活动，或者是 10-130 级的燃烧加速器活动），不妨第一个燃烧角色选择杰诺（菈菈的清怪能力很强，如果对游戏上手不是很了解，可以先练菈菈）。
      如果是 大斗燃（10-200 级的极限燃烧活动），请 不要 选择杰诺为燃烧角色！！！因为杰诺并不是新号最好的起步职业。

TIP

如果不小心创建了燃烧角色，您可以删除此角色，等 24小时 后再次确认删除，就可以重新创建新的燃烧角色了。

  • 燃烧加速器活动：燃烧等级：10 - 130，没有宠物，又名：小斗燃
  • 超级燃烧活动：燃烧等级：10 - 150，宠物时长 5 小时，又名：小斗燃
  • 极限燃烧活动：燃烧等级：10 - 200，宠物时长 25 小时，又名：大斗燃

（注：请记得从杂货店购买宠物食品，以免宠物饿死。宠物时长只计算宠物使用时间，不用时可以召回宠物以暂停时长）

小结一下：

  • 当前活动是小斗燃（10-150 等级的超级燃烧，或者是 10-130 等级的燃烧加速器），就选择 菈菈 或者 杰诺 。
  • 当前活动是大斗燃（10-200 等级的极限燃烧），在不知道自己玩什么职业为大号的情况下，可以创建 阴阳师。
  • 想燃烧角色留给大号使用，可以先练 幻兽师、恶魔复仇者 。

# 2.2 开启联盟方阵，越来越强大

联盟方阵（联盟战地系统）的开启，是需要满足条件才能开启的，目前开启的条件有两个：

  • 所有角色等级和大于 500 等级（注：仅限进行 2转 60 级后的角色或 130级 的神之子，才能进行联盟等级计算）
  • 有一个角色等级大于 200 等级（注：需要完成 5转）

前期的职业，不考虑燃烧活动的情况下，幻兽师、恶魔复仇者、菈菈、阴阳师，这四个职业建议先升级到 140 级（虽然 120 级就有 2级 link传授，但是 140 级有 S级 的联盟加成），这样就能有 500+ 等级，可以开启联盟方阵啦。

TIP

联盟方阵前期摆放属性建议：

物理职业：优先 HP，再者对应职业的属性（特殊的请选择攻击力，如恶魔复仇者、杰诺）

法师职业：优先 HP、MP，再者对应职业的属性

后续不知道练什么职业，可以看看高手篇。

学习成长，自成高手。

# 三、高手篇

作为对 MapleStory 有自己独特的见解，高手们往往会优先定位符合大号职业的 Link传授及联盟加成。

而高手们对游戏的各路玩法都非常熟悉，往往会租借（或购买）通用练级装备或道具，并且会懂得如何高效升级练功方法。

由于本文属于新手入门教程，在这先大概撰写一下高手们是如何定位小号练级顺序，以达到他们心中所期的 link传授 及 联盟加成。

TIP

高手们一般最高优先级是先练大号（本尊）到 200 或 210，之后就会停下大号的冲等，主要目的是每天完成奥术之河日常任务，以提升 ARC（秘法符文）。

# 3.1 高效练级，从经验加成开始

作为对游戏理解深刻的玩家，往往会非常重视经验加成，这可是能缩短练功时间的，为此，高手们开局往往会优先安排下面职业：

  • 神之子：

    当前版本如果能创建神之子职业，那么优先级是先练一个神之子，毕竟神之子提供时间短暂，提供时期长达每两年左右提供一次。而其 200等级 时的 SS级 联盟角色卡，带来账号内所有角色享受增加 +10% 的获取经验加成，同时，Link传授能减伤 15%，对前期开荒来说非常实用。
    唯一缺点，神之子特殊的剧情原因，导致其解锁 Link传授 和 SS级 联盟角色卡，比较繁琐，在不熟悉剧情的情况下，可能会比其他职业到达 200等级，花费更多的时间。

  • 精灵游侠（CMS国服又名：双驽精灵）：

    前期精灵游侠，等级只需要到达 140，就能满足到后期了，120 级有 2 级 Link传授，拥有增加 +15% 的获取经验加成。而有些高手们可能计划是一天练四个 140 级的小号，那么，他们往往会练多一个 120 级（或 140级 多占一个联盟格子） 的精灵游侠，以达到能够每天四个小号享受到 +15% 的经验加成。（韩服已经改版，每个角色每天最多可以传授 10 次，前四次免费）

  • 龙魔导士（CMS国服又名：龙神）：

    龙魔导士的 Link传授，非常实用，它能提高符文之轮的 BUFF 时间，让原本 3分钟的轮之力经验加成 BUFF 达到 4分半 的持续时间。别小看这点时间加成，前期小号练级在经验拉满的情况下，升一级是按秒算的。例如：在鸟2那里，释放符文之轮的轮之力，100级前，有的职业能平均三秒升一级。

  • 狂狼勇士（CMS国服又名：战神）:

    狂狼勇士的 Link传授，能提高 +650% 的连击宝珠经验，在前期一个连击宝珠可是能直接升一级的，能提高不少的效率，百级后连击宝珠的经验提升存在感减少，聊胜于无。但是后期是在固定一个地图练级时，能达到很高的连击数，提升非常可观。

小结一下：

  • 除神之子之外，另外三个都是只需要达到 120 级 就能传授 2级 Link 给其他角色，因为都是有练级套，可以无压力达到 140 级，拿到 S级 联盟角色卡，能多占一格，增加一点联盟属性加成非常实用。
  • 神之子由于剧情特殊性，如果不想浪费太多时间，可以创建一个神之子放着，后续有时间再来把神之子提升到 200 等级。

# 3.2 更高效的练级，从秒怪开始

我们所说的秒怪，就是使用一次攻击技能能直接把怪物打死，甚至严格一些，技能产生的附加攻击也能把怪物打死，这种瞬间击杀称为秒怪。

先来做一个不是很严谨的比较：在同等经验加成条件下，能秒怪升一等级假设需要 10分钟，那么需要出手两次才能击杀怪物，所需要升一等级的时间就是 20分钟，同理，需要出手次数更多，则升一等所需要的时间就更多。

从这个不是很严谨的比较可以看出，能秒怪，真的是高效练级的前提条件。

而新枫之谷这游戏，练级往往会分下面几大类：

  • 达到 140等级，获取 Link传授 及 S级 联盟角色卡

    （难度小，练级装备成本低）

  • 达到 200等级，获取 Link传授 及 SS级 联盟角色卡

    （难度中，练级装备需分各大职业群，装备多，成本高）

  • 达到 250等级，获取 Link传授 及 SSS级 联盟角色卡

    （难度大，传说级别的练级装备是基础，高星是必要，还需要做神秘河的神秘压制，成本太大，实在是高上之高）

而这几类里，难度小，又能快速提升加成的，首选 Link传授。

为此，需要靠前安排这些职业。

提升伤害的 Link传授 职业（建议先练到 140 等级）

  • 菈菈

    Link传授增加 5% 伤害，击杀 20 个怪物再增加 11% 怪物伤害。可以说增加 16% 伤害了，因为持续结束后击杀 20 个怪物应该用不了多少秒的时间。

  • 阴阳师

    Link传授增加 10% 伤害

  • 恶魔复仇者

    Link传授增加 10% 伤害

  • 杰诺（CMS国服又名：尖兵）

    Link传授增加 10% 全属性，S级联盟角色卡增加力量、敏捷、运气：+20

  • 剑豪

    Link传授增加全属性 +25，增加攻击力 +15，S级联盟角色卡增加 3% 爆击伤害

  • 亚克（CMS国服又名：影魂异人）

    Link传授增加保持战斗状态伤害，最高增加 11% 伤害，S级联盟角色卡增加力量 +40

  • 伊利恩（CMS国服又名：圣晶使徒）

    Link传授每移动一定距离增加伤害，最高增加 12% 伤害，S级联盟角色卡增加智力 +40
    如果是一直站着输出的职业，加成效果可能并不理想。

除了上面的职业，还有一些职业的 Link传授，也是增加伤害，如：

  • 虎影

    对满血怪物增加 14% 伤害（增加后能秒怪就是利器，不能秒就没什么用了）

  • 凯殷

    每击杀 8 只怪物完成一次准备，准备五次之后（击杀 40 只怪）增加 17% 伤害，（前期准备时间长，激活后持续 20 秒，冷却要 40 秒，不能常态保持，不是很理想）

  • 冒险家的三个海盗

    三个海盗职业的 Link传授 能叠加一起使用，只占用一个 Link传授位置，达到满级 6级 的 Link传授 加成：
    全属性 +70，HP、MP +1225，减伤 15%
    如果需要这些属性加成，由于需要练三个职业到 140级，加上他们并没有良好的范围清怪技能，相比前面只需要练一个职业来说，性价比不高，建议在前面需要靠前安排的职业练完后，再来安排这三个职业（分别是：枪神、拳霸、重炮指挥官）

  • 一些增加异常状态伤害等，受于文章篇幅，加上也不是很适应大部分职业，在这就略过

# 四、寻找自己所喜欢的职业作为大号

经过前面这些前期通用提升加成，接下来就可以为自己的大号职业，进行冲击更高等级了。同时，也可以定制专属的 Link传授 及 联盟角色卡 加成，以进一步提升。
