# 一、Link 传授

Link 传授，可以理解为每个职业都有一个特有的技能属性，传授给其他角色，让这个角色，也拥有这个特有技能的属性加成，在使用上，有以下限制：

  • 1、每个角色最多只能使用 12 种其他职业的 Link 传授（不包括自身职业）

  • 2、接收 Link 传授 的角色，不能重复接收同一职业的 Link 传授

    比如，我想 12 个 Link 传授 都只接收阴阳师的传授以达到增加 +120% 伤害。这是行不通的！

  • 3、Link 传授 的技能等级受其传授角色的等级限制，几乎所有职业在 70级 时解锁 1级 Link传授，120级 时解锁 2级 Link 传授，210级 时解锁 3级 Link 传授（3级 Link 传授 仅 GMS 服部分职业开放）

    神之子比较特殊，需要完成所有剧情，才能解锁最高级 Link 传授（5级）

  • 4、每个角色的 Link 传授 每天最多传授 两次 给其他职业

    如果您想当天同职业的 Link 传授 大于两次以上传授给其他职业，您可以多练一个这个职业的角色，以达到 2次/天 x 2个同职业传授 = 4次同职业传授/每天，同理，更多就再多准备一个

  • 5、部分职业群的 Link 传授 能叠加使用，只占一个 Link 传授 位置，但是不能接收多个同一职业进行叠加

    比如：冒险家的三位魔法师，叠加后的 Link 传授 有 6级，只能接收火毒魔导师、冰雷魔导师、主教这三个一起进行叠加，不能接收三个主教的传授来一起进行叠加

  • 6、只有接收传授的职业才能享受到传授加成，如果传授的角色被另一个职业接收传授，那么，之前的职业将不会享受到传授加成。
