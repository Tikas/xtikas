# ❓ 解决聊天打字全是问号

很多萌新可能会好奇，在新枫之谷游戏里打字会是 “？” 号显示，这是由于台服冒险岛对简体文字支持不友好导致的，请大家切换繁体输入再来交流哦！

TIP

如果切换繁体依旧还是有问号显示，您可以下载一个思源黑体作为输入法的默认显示字体。

思源黑体官方 Github 下载页面

如果您的系统是 windows 10+ ，您可以尝试下载 OTC 版本：Static Super OTC（OTC 版本已经拥有所有字）

如果 OTC 版本并不能良好支持您的系统（如 win 7）和输入法（某些输入法不支持显示字体更改设置），请下载 OTF 版本，最好是直接下载 CN 版本的：Language Specific OTFs Simplified Chinese (简体中文)（CN 版本多数会包含繁体，这样不会出现缺字现象，当然，如果您觉得还是不够保守，那就下载繁体的 OTF）

# 🆘 更新 64 位应用程序后打字乱码

MapleSotry 全面启用 64 位应用程序，如果您没有使用繁体区域语言，此时，您是使用转区工具来启动游戏的。

由于 LE 并不支持 64 位应用程序，现在所使用的转区工具是 InWILL 开发的 Locale Remulator。

如果您只是打字是乱码显示，那么，您应该把第三方缤放登录器更新到 4.3 版本。

如果您是复制内容是乱码，那么，请等待 InWILL 开发者对此问题进行修复。

# 二、五笔输入法设置

由于大多五笔对 win10 系统支持不太好友，推荐使用 “精灵五笔” ，由于他们都是相同的源码（反正怎么看都是一样的，先打上一个相同源码吧），其设置方法应该是一样的，本文以 “花儿五笔” 为例。

推荐大家使用 精灵五笔，由于 花儿五笔 新版本进行大量更改，导致非常不好用！

打开精灵五笔设置程序，在常用这里，把初始状态的输入开关选择 “关闭（英文）”，这样可以把其他输入法都去掉，默认就只有一个五笔，同时，默认是不打字的英文字母输入，我非常喜欢这样。

在需要打字时，只需要按一下 左边的 Shift 键就可以了。请注意，由于本文一会要把右边的 Shift 设置为繁体切换，所以，请大家养成以左边 Shift 键为中英文切换，常用里设置如下图：

接下来，就是设置 右边 Shift 键为简繁体切换啦。同样在精灵五笔设置，在高级选项，点击 “按键功能” 打开快捷键配置程序，（这里要注意，请不要勾选前面的 “禁用按键组合功能定义” ，不然设置会失效！），如下图：

在按键配置程序里的右边 Shift 键，弹出的菜单里选择：扩展功能->简体码表转繁体 ，完成设置保存之后，到此，就设置完成啦！

默认程序为打英文，要打中文时，按一下左边的 Shift，就是中文了，如果需要打繁体，再按一下右边的 Shift 键，就是繁体输出啦，不想繁体时，再按一下 右边的Shift，就是简体啦，非常方便，只需要按一个键就可以搞定！

# 三、补充相关

如果你在游戏里想打：爲什麼？，会发现繁体的 “爲” 字显示是问号，你需要为这个繁体的 “爲” 字设置一下自定义短语，这样会好些，正确能显示的为什么是这样的：

简体：为什么？ 繁体：爲什麼？（游戏里显示：？什麼） 正确繁体：為什麼？（游戏里显示：為什麼）

“為” 的五笔打法：etno 或者 ytno

还有一些常见词，比如做着什么事的 “着” 也是显示不出来的，其实是要打成：“著” 就可以了，更多的自己网上找找吧，这里就不再补充啦~
